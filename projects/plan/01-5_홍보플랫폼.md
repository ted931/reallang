## 5. 무료 홍보 플랫폼 기획

### 5.1 핵심 전략 요약

> **한 줄 정의**: 제주도 자영업자에게 **무료 홍보 페이지**를 제공하고, 이 페이지들의 SEO 파워를 모아 **제주패스 도메인 권위**를 높인 뒤, **카페패스** 구독 상품으로 수익화한다.

```
[자영업자 무료 가입] → [가게 홍보 페이지 자동 생성]
    → [SEO 최적화 콘텐츠 수천 페이지]
        → [네이버/구글 상위 노출]
            → [여행객 유입 → 렌터카/카페패스 전환]
```

### 5.2 사업자 인증 체계

| 단계 | 내용 | 소요 시간 | 자동화 |
|------|------|----------|--------|
| 1. 전화번호 인증 | SMS 인증코드 | 30초 | 100% |
| 2. 사업자등록번호 | 국세청 API 실시간 검증 | 5초 | 100% |
| 3. 대표자 본인확인 | PASS 인증 or 공동인증서 | 1분 | 100% |
| 4. 가게 정보 입력 | 상호명, 주소, 카테고리, 사진 | 3분 | AI 보조 (네이버 크롤링 프리필) |
| 5. 관리자 승인 | 중복/부적절 필터링 | 24시간 이내 | 자동 90% + 수동 10% |

**남용 방지 규칙**:
- 동일 사업자번호 중복 가입 차단
- 폐업 사업자 자동 필터
- AI 기반 가짜 리뷰 탐지 (자기 리뷰 등)

### 5.3 SNS 콘텐츠 자동 생성

**자동 생성 콘텐츠 유형**:

| 유형 | 생성 트리거 | 출력물 | 사장님 액션 |
|------|-----------|--------|-----------|
| 가게 소개 카드뉴스 | 가입 완료 시 | 인스타 정사각 이미지 4장 | 다운로드 → SNS 업로드 |
| 메뉴/상품 하이라이트 | 메뉴 등록 시 | 인스타 스토리 템플릿 | 1탭 공유 |
| 리뷰 감사 카드 | 별점 4+ 리뷰 수신 시 | 리뷰 인용 이미지 | 자동 or 수동 공유 |
| 오늘의 추천 | 매일 10:00 | "오늘의 제주 카페" 피드 | 태그 시 노출 |
| 시즌 프로모션 | 시즌 시작 2주 전 | 프로모션 배너 + 카피 | 편집 후 게시 |

**AI 이미지 생성 스펙**:
- 입력: 가게 사진 3-5장 + 메뉴 텍스트 + 분위기 키워드
- 출력: 브랜드 컬러 반영 카드뉴스 (1080x1080, 1080x1920)
- 모델: Stable Diffusion XL (배경) + 포토 합성
- 처리 시간: 30초 이내
- 월 무료 생성: 20장 / 프리미엄: 무제한

### 5.4 타겟 페르소나

#### 페르소나 A: 사장님 (공급측)

| 속성 | 상세 |
|------|------|
| 이름 | 김제주 (45세, 남) |
| 업종 | 애월 카페 운영 3년차 |
| 월 매출 | 2,800만원 (순이익 700만원) |
| 고민 | 네이버 플레이스 리뷰 관리 벅참, 인스타 할 시간 없음, 블로그 마케팅 비용 부담 |
| 디지털 역량 | 카카오톡/인스타 기본 사용, 블로그 글쓰기 어려움 |
| 기대 | "그냥 사진 올리면 알아서 홍보해주면 좋겠다" |
| 지불 의향 | 무료면 무조건 쓰겠다, 효과 있으면 월 5만원까지 가능 |

#### 페르소나 B: 여행객 (수요측)

| 속성 | 상세 |
|------|------|
| 이름 | 이서울 (32세, 여) |
| 여행 빈도 | 제주 연 2회, 주로 커플/친구 |
| 계획 방식 | 인스타 탐색 → 네이버 검색 → 블로그 비교 → 예약 |
| 고민 | 너무 관광지만 나옴, 로컬 맛집 찾기 어려움 |
| 기대 | "현지인이 추천하는 숨은 가게를 한 번에 보고 싶다" |
| 행동 특성 | 리뷰 사진 많은 곳 선호, 동선 최적화 중요시 |
