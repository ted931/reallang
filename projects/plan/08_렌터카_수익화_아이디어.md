## 전체 10개

### 1. AI 차종 추천 (MyCarPick) ★★★★★
- 여행 인원/목적/짐 → AI가 최적 차종+가격 추천
- 업셀링 유도 (소형→SUV), 전환율 상승
- **3~4주**, AI 플래너 기술 재활용

### 2. 날씨 드라이브 코스 (WeatherDrive) ★★★★★
- 렌터카 이용 중 날씨 변화 → 실시간 코스 자동 조정
- "비 오면 실내 코스 추천" → CafePass 연계
- **3~4주**, 기존 웨더맵+코스메이커 합체

### 3. 스마트 주유 (Smart Fuel) ★★★★★
- 반납 전 최저가 주유소 안내 + 필요 주유량 계산
- 연료 분쟁 제로 → CS 비용 대폭 절감
- **3~4주**, 주유소 공공데이터+카카오맵

### 4. 제주 주차 파인더 (ParkEasy) ★★★★★
- 관광지 실시간 주차 가용 정보 + 예약
- 성수기 주차 문제 해결 → 앱 DAU 급상승
- **6~8주**, 공영주차장 공공데이터

### 5. 로열티 프로그램 (제주마일) ★★★★★
- 이용거리/횟수/리뷰 마일리지 → 등급별 혜택
- 리피터 록인 → 가격 경쟁 탈피
- **5~6주**, Supabase DB

### 6. 실시간 도로상황 (Road Pulse) ★★★★☆
- 제주 특화 교통정보 + AI 우회경로
- "제주패스로 빌리면 길 안 막힘" 포지셔닝
- **5~6주**, 교통 API 연동

### 7. AI 사고 대응 (ClaimHelper) ★★★★☆
- 사고 시 현장 가이드 + 보험 서류 자동 작성
- 고객 패닉 순간 → 평생 고객 전환
- **6~8주**, 보험사 제휴 필요

### 8. B2B 위젯 (PlugCar) ★★★★☆
- 블로거/여행사 사이트에 예약 위젯 임베드
- 롱테일 유입 채널 확보, 어필리에이트 수수료
- **4~5주**, 예약 API 래핑

### 9. 다이나믹 프라이싱 (SmartPrice) ★★★☆☆
- 수요/날씨/항공편 기반 가격 실시간 최적화
- 매출 극대화 + 유휴 차량 감소
- **8~10주**, 정밀 튜닝 필요

### 10. 무인 픽업/반납 (TapGo) ★★★☆☆
- QR+앱 기반 대면 없이 3분 만에 픽업
- 혁신적 UX, 인건비 절감
- **12~16주**, 하드웨어(스마트 키박스) 필요
