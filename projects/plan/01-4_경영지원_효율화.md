### 4.3 회계 자동화

#### 4.3.1 현재 회계 플로우 문제점

| 문제 | 빈도 | 영향 | 해결책 |
|------|------|------|--------|
| 영수증 분실 | 월 15-20건 | 매입세액 공제 누락 (연 800만원 추정) | 모바일 촬영 즉시 업로드 |
| 계정과목 오분류 | 월 30건+ | 재무제표 왜곡 | AI 자동 분류 |
| 정산 지연 | 파트너 50개사 | 파트너 불만, NPS 하락 | 자동 정산 워크플로 |
| 카드 사용 통제 | — | 비인가 지출 월 200만원+ | 법인카드 한도/카테고리 제한 |

#### 4.3.2 영수증 관리 시스템

**자동 수집 채널**:

| 채널 | 방식 | 커버리지 |
|------|------|---------|
| 법인카드 | 카드사 API 자동 수집 | 85% |
| 모바일 촬영 | OCR → 자동 입력 | 10% |
| 이메일 포워딩 | 세금계산서 자동 파싱 | 4% |
| 수기 입력 | 최후 수단 | 1% |

**OCR 정확도 목표**: 98.5% (핵심 필드: 일자, 금액, 사업자번호, 항목)

**처리 플로우**:
```
영수증 수집 (자동/촬영)
  → OCR + AI 계정과목 분류
    → 직원 확인 (10초 이내)
      → 팀장 승인 (자동 규칙 내 건은 자동 승인)
        → 회계 장부 자동 반영
          → 월말 부가세 자동 집계
```
