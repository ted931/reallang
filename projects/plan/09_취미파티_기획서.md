# 제주 취미 파티 — 혼자 온 여행자를 위한 동행 매칭

> **한 줄 요약**: "나 렌트 있는데 같이 자전거 타러 갈 사람?" — 취미별 파티장이 모집하고, 솔로 여행자가 참여하는 동행 플랫폼

---

## 1. 왜 이걸 만들어야 하는가

### 페인 포인트
- 제주 방문자 중 **1인 여행 비율 약 25%** (제주관광공사 2024)
- 혼자 왔는데 **자전거 타기, 등산, 서핑, 낚시** — 혼자 하기 심심하거나 불안
- 렌터카 빌렸는데 혼자 쓰기엔 비용이 아깝고, 동선도 비효율
- "같이 갈 사람" 검색하면 → 네이버 카페, 에브리타임 → **신뢰도 낮고 매칭 어려움**

### 우리가 풀 수 있는 이유
- **제주패스가 이미 렌터카/액티비티/숙소를 가지고 있음**
- 파티장이 "내가 렌터카 있어, 엔빵하자" → **제주패스 렌터카 예약과 직결**
- 취미별 코스를 **AI 코스 메이커로 자동 생성** 가능
- 날씨 연동 → "오늘 날씨 좋으니까 같이 올레길 걷자" 자동 추천

## 2. 핵심 개념

### 파티장 (모집자)
```
"나는 내일 애월에서 자전거 타려고 하는데
 렌트 카니발 있고, 자전거 2대 실을 수 있어.
 같이 갈 사람 2명 모집! 비용은 렌트비+점심 엔빵"
```

### 파티원 (참여자)
```
"오 자전거 타고 싶었는데! 참여합니다.
 저 자전거 없는데 빌릴 수 있나요?"
```

### 파티 유형

| 카테고리 | 예시 | 필요한 것 |
|---------|------|----------|
| 🚴 자전거 | 애월 해안도로 라이딩 | 자전거 (렌탈 가능), 렌터카 |
| ⛰️ 등산/트레킹 | 한라산 영실코스, 오름 투어 | 등산화, 교통 |
| 🎣 낚시 | 서귀포 방파제 낚시, 선상 낚시 | 장비 (렌탈 가능) |
| 🏄 서핑 | 중문 서핑 레슨 | 서핑보드 (샵 제공) |
| 🏃 러닝/워킹 | 올레길 7코스 걷기 | 운동화 |
| ☕ 카페탐방 | 애월 카페거리 5곳 투어 | 렌터카 or 택시 |
| 📸 출사 | 일출/일몰 포토스팟 투어 | 카메라 |
| 🍊 맛집투어 | 흑돼지+해물+디저트 코스 | 렌터카, 빈 속 |
| 🤿 스노클링/다이빙 | 우도 바다 체험 | 장비 (샵 제공) |
| 🎨 공방체험 | 도자기, 향수, 가죽 만들기 | 예약 |

## 3. 사용자 플로우

### 파티장 (모집)
```
① 취미 카테고리 선택 (자전거, 등산, 카페...)
② 일정 입력 (날짜, 시간, 장소, 코스)
   → AI 코스 추천: "이 취미에 맞는 코스 3개" 자동 제안
③ 조건 설정
   - 모집 인원: 2~6명
   - 비용 분담: 엔빵 / 무료 / 정해진 금액
   - 렌터카 여부: 있음(동승 가능) / 없음(각자 이동)
   - 장비: 있음 / 렌탈 필요
④ 파티 게시 → 카카오톡/링크 공유 가능
```

### 파티원 (참여)
```
① 피드에서 파티 검색 (날짜, 취미, 지역 필터)
② 파티 상세 확인 (코스, 비용, 파티장 프로필)
③ "참여 신청" 클릭
④ 파티장 승인 → 카카오톡 단톡방 자동 생성
⑤ 당일 만남 → 활동 → 후기 작성
```

### 매칭 후
```
┌────────────────────────────────────────────┐
│  🚴 애월 자전거 라이딩  내일 10:00         │
│                                            │
│  파티장: 김제주 (⭐4.8, 파티 12회)          │
│  코스: 애월항 → 곽지해변 → 한림항 (25km)   │
│  현재 날씨: 맑음 22° 🌤️                    │
│                                            │
│  🚗 렌터카 있음 (카니발, 자전거 3대 가능)   │
│  💰 비용: 렌트비+점심 엔빵 (인당 약 3만원)  │
│  👥 모집: 2/4명 참여 중                     │
│                                            │
│  [참여 신청]  [코스 상세보기]  [채팅하기]    │
└────────────────────────────────────────────┘
```

## 4. 제주패스 매출 연결 (이게 핵심)

| 연결 포인트 | 어떻게 | 예상 수익 |
|------------|--------|----------|
| **렌터카 예약** | 파티장이 "렌터카 필요" → 제주패스 예약 연결 | 건당 수수료 |
| **렌터카 공유** | "엔빵" 시 파티원도 렌터카 결제 참여 | 추가 예약 유도 |
| **액티비티 예약** | 서핑 레슨, 스노클링, 공방 → 제주패스 제휴 업체 | 판매 수수료 15% |
| **장비 렌탈** | 자전거, 낚싯대, 서핑보드 → 제휴 렌탈샵 | 중개 수수료 10% |
| **카페패스** | 카페탐방 파티 → 카페패스 묶음 판매 | 자체 상품 |
| **숙소** | "1박 포함 등산 파티" → 숙소 예약 | 건당 수수료 |
| **프리미엄** | 파티장 인증 배지, 상위 노출, 무제한 모집 | 월 구독 ₩9,900 |

### 예상 매출 시나리오 (6개월 후)
- 월 활성 파티: 300건
- 파티당 평균 참여자: 3명
- 렌터카 연결율: 40% → 120건 × 평균 ₩80,000 × 수수료 12% = **₩1,152,000/월**
- 액티비티 연결: 30% → 90건 × ₩40,000 × 15% = **₩540,000/월**
- 카페패스 판매: 200건 × ₩15,000 × 마진 40% = **₩1,200,000/월**
- 프리미엄 구독: 50명 × ₩9,900 = **₩495,000/월**
- **합계: 약 ₩3,400,000/월** (초기)

## 5. 기존 프로젝트와의 시너지

```
날씨 지도 ──→ "오늘 맑은 곳에서 파티 모집 중" 배너
AI 코스   ──→ 파티장이 코스 자동 생성
지도       ──→ 파티 위치 핀으로 표시
플래너    ──→ 파티 일정을 여행 일정에 추가
챗봇      ──→ "혼자인데 같이 할 사람 있나요?" → 파티 추천
```

## 6. 경쟁 대비 차별점

| 기존 | 문제 | 우리 |
|------|------|------|
| 네이버 카페 "제주 동행" | 글 올리고 기다리기, 신뢰 없음 | 프로필+후기+실시간 매칭 |
| 에브리타임 | 학생 한정, 제주 특화 아님 | 제주 전용, 취미별 카테고리 |
| 소모임 앱 (프렌트립 등) | 제주 컨텐츠 부족, 가격 비쌈 | 무료 모집, 엔빵 구조 |
| 카카오톡 오픈채팅 | 검색 불편, 필터 없음 | 날짜/취미/지역 필터 + AI 코스 |

**핵심 차별**: 제주패스의 렌터카/액티비티/숙소와 연결 = **"같이 놀자"에서 "같이 예약하자"까지 원스톱**

## 7. MVP 스펙 (2~3주)

### 필수 기능 (P0)
- [ ] 파티 생성 (취미, 날짜, 장소, 인원, 비용)
- [ ] 파티 피드 (날짜/취미/지역 필터)
- [ ] 파티 상세 (코스, 비용, 파티장 정보)
- [ ] 참여 신청 + 파티장 승인
- [ ] 간단 프로필 (닉네임, 취미 태그, 한줄 소개)

### 있으면 좋은 (P1)
- [ ] AI 코스 자동 추천 (코스 메이커 연동)
- [ ] 날씨 연동 ("오늘 맑은 곳" 파티 상위 노출)
- [ ] 카카오톡 공유 (OG 이미지 자동 생성)
- [ ] 후기/평점 시스템

### 추후 (P2)
- [ ] 카카오 소셜 로그인
- [ ] 실시간 채팅 (Supabase Realtime)
- [ ] 렌터카/액티비티 예약 연결
- [ ] 파티장 프리미엄 구독

## 9. 왜 이게 우선순위가 높아야 하는가

1. **AI 도구는 도구일 뿐** — 사용자가 반복 방문할 이유가 없음. 코스 한번 만들면 끝.
2. **커뮤니티는 리텐션** — "다음에 또 같이 하자" = 재방문, 재예약
3. **제주패스 핵심 사업과 직결** — 렌터카 공유, 액티비티 예약이 곧 매출
4. **UGC(사용자 생성 콘텐츠)** — 파티 후기가 자연스러운 마케팅 콘텐츠
5. **네트워크 효과** — 사용자가 늘수록 파티가 다양해지고 가치 증가
