## 1. 제주 실시간 날씨 레이어

### 1.1 개요

제주도는 동서남북 기후 편차가 크고, 산간/해안 기온차가 5-10도에 달한다. 기상청 AWS(자동기상관측장비) 28개소 데이터를 실시간으로 지도 위에 시각화하고, 날씨 조건에 따라 자동으로 코스를 전환 추천한다.

### 1.2 데이터 소스

| 소스 | 갱신주기 | 항목 | API |
|------|----------|------|-----|
| 기상청 초단기실황 | 매시 정각 | 기온, 강수량, 풍속, 습도 | 공공데이터포털 REST |
| 기상청 동네예보 | 3시간 | 3일 예보, 강수확률 | 공공데이터포털 REST |
| 제주 AWS 28개소 | 1분 | 기온, 풍향풍속, 강수 | 기상자료개방포털 |
| 해양기상부이 | 10분 | 파고, 수온, 풍속 | 기상청 해양기상 API |
| 제주관광공사 CCTV | 실시간 | 주요 관광지 영상 | 제주관광정보센터 |
| UV지수/미세먼지 | 1시간 | 자외선, PM2.5, PM10 | 에어코리아 API |

### 1.4 날씨별 자동 코스 추천 로직

| 날씨 조건 | 자동 전환 | 추천 카테고리 | 예시 장소 |
|-----------|----------|--------------|----------|
| 맑음 + 기온 20-28° | 실외 우선 | 올레길, 뷰포인트, 해변 | 월정리해변, 사려니숲길 |
| 맑음 + 기온 10° 미만 | 실외(방한) | 등산, 러닝, 온천 | 한라산 영실코스, 중문온천 |
| 비 (강수 1-10mm/h) | 실내 전환 | 카페, 박물관, 체험 | 아쿠아플라넷, 유리의성 |
| 폭우 (강수 10mm+ ) | 완전 실내 | 맛집, 실내체험, 숙소근처 | 흑돼지거리, 제주맥주양조장 |
| 강풍 (풍속 14m/s+) | 산간 회피 | 해안 저지대, 실내 | 이호테우해변 회피 → 실내카페 |
| 파고 2m+ | 해양 금지 | 서핑/다이빙 취소 알림 | 대체: 실내 서핑풀 |
| 안개 (시정 1km 미하) | 산간 회피 | 해안 드라이브, 카페 | 1100고지 회피 → 해안도로 |
| UV지수 8+ | 그늘 코스 | 숲길, 동굴, 실내 | 비자림, 만장굴 |

### 1.5 자동 전환 알고리즘

```
function autoSwitchCourse(currentCourse, weather) {
  // 위험도 점수 산출
  riskScore = calcRisk(weather.rain, weather.wind, weather.visibility)
  
  if (riskScore >= 80) {
    // 긴급 전환: 완전 실내 코스로 교체
    return findIndoorAlternative(currentCourse, radius=5km)
  } else if (riskScore >= 50) {
    // 부분 전환: 실외 스팟만 실내로 교체
    return replaceOutdoorSpots(currentCourse, weather)
  } else if (riskScore >= 30) {
    // 경고만 표시, 우비/방풍 준비물 안내
    return addWeatherWarning(currentCourse, weather)
  }
  return currentCourse  // 변경 없음
}
```
