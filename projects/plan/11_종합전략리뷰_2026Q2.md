## 1. 현재 상태 요약

### 프로젝트 현황 (16개)

**통합 서비스 (TOP)**
| 서비스 | 통합 대상 | 상태 | 포트 |
|--------|-----------|------|------|
| 제주 여행 (jeju-travel) | weather+map+planner+course+drive | BETA | 3020 |
| 제주 렌터카 (jeju-car) | car-pick+smart-fuel+cost | 준비중 | 3021 |
| 제주 비즈니스 (jeju-biz) | promo+dashboard+chatbot | 준비중 | 3022 |

**기존 프로토타입 (13개)** — 포트 3001~3013

- Next.js 15/16, React 19, Tailwind CSS v4
- AI: OpenAI GPT-4o / GPT-4o-mini
- DB: Supabase (PostgreSQL)
- 지도: Leaflet + OpenStreetMap
- 날씨: 기상청 API (공공데이터포털)

## 2. 핵심 문제 3가지

### P0: 통합 서비스 배포 누락
- jeju-travel, jeju-car, jeju-biz가 PM2/nginx에 미등록
- 포탈에서 "여행 시작하기" 클릭 → 실서버에서 404
- **해결**: PM2 ecosystem.config.js + nginx.conf + deploy.yml 업데이트

### P0: 결제 기능 0개
- 어떤 프로젝트도 돈을 받지 못하는 상태
- 마스터플랜 800M 수익 전제가 무의미
- **해결**: 아임포트/토스페이먼츠 결제 게이트웨이 통합 필요

### P1: 프로젝트 간 데이터 사일로
- 13개 프로젝트가 동일 API를 각자 호출
- API 쿼터 위험 + 사용자 경험 단절
- **해결**: Supabase 공유 캐시 레이어 + sessionStorage 크로스탭 공유 (일부 구현 완료)

## 3. 수익화 전략 — 현실 점검

### 마스터플랜 가정 vs 현실

| 항목 | 기획서 가정 | 현실 |
|------|-----------|------|
| SEO 월 트래픽 | 30,000명 | ~2,400명 (한국 외부 검색 점유율 8%) |
| 카페 파트너 | 6,000곳 | 제주 전체 카페 3,800곳 |
| 카페패스 전환율 | 13% | 현실적 3~5% |
| M18 매출 | 87.5억 | 현실적 18~30억 (보수적) |
| 여행객 CAC | 2,800원 | 실제 8,500원대 (유료 광고 포함) |

### 수익화 우선순위

**1순위: jejupass-promo → 카페패스** (가장 빠른 수익원)
- 자영업자 무료 등록 → 여행객 유입 → 카페패스(5잔 쿠폰) 판매
- 4주 검증: 애월/동문 카페 50곳 인터뷰 → 70% 이상 "무료면 등록" 확인 시 Go
- 필요: 결제 게이트웨이, Supabase 사업자 테이블
- 보수적 M12: 월 2.3억

**2순위: jeju-party → 참여비 결제**
- Supabase 이미 연동됨 (개발 속도 가장 빠름)
- 파티 참여비 선결제 + 렌터카 번들 할인
- 보수적 M12: 월 1.5억

**3순위: 렌터카 CTA → 기존 매출 증대**
- 모든 AI 결과물(코스/플래너)에 "렌터카 예약" 버튼 삽입
- 직접 수익이 아닌 기존 예약 전환율 향상 (+2~3%)

## 5. 사용자 확보 전략

### 현실적 채널 우선순위

| 채널 | 예상 효과 | 비용 | 타이밍 |
|------|---------|------|--------|
| "제주 날씨" SEO 랜딩 | 월 2,000+ UV | 무료 | 즉시 (이미 구현) |
| AI 코스 → OG 이미지 공유 | 카카오톡/인스타 바이럴 | 무료 | 2주 |
| 카페 파트너 → 오프라인 QR | 가게당 방문 30+/월 | QR 인쇄비 | 4주 |
| 네이버 블로그 자동 생성 | 월 5,000+ UV (장기) | 무료 | 8주 |

### 핵심 인사이트
- AI 도구(코스/플래너)는 **1회성 사용** → 리텐션 낮음
- **파티(커뮤니티)가 유일한 재방문 동력**
  - 파티 → 리뷰 → UGC → SEO → 신규 유입 → 파티 참여 사이클
  - 파티 운영자 프리미엄 구독(9,900원/월) = 안정적 반복 수익
