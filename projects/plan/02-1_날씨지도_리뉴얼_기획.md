# 제주 날씨 지도 — 리뉴얼 기획서

> **문제**: 현재 날씨 지도는 "기온/강수 정보 나열"에 그침. 여행자가 왜 이걸 봐야 하는지, 제주패스에 어떤 매출이 연결되는지 불분명.
>
> **핵심 질문**: 날씨 데이터로 뭘 해야 제주패스 고객이 돈을 쓰게 되는가?

---

## 1. 포지셔닝: "날씨 때문에 망하는 제주여행을 구한다"

### 제주 여행의 핵심 Pain Point
- 제주는 **동서남북 날씨가 다름** — 서귀포 비, 제주시 맑음이 일상
- 여행객의 78%가 "날씨 때문에 일정 변경 경험" (제주관광공사 2024)
- 비 오면 뭐 하지? → **검색 폭증하는 순간이 곧 전환 기회**

### 우리의 답
> **"지금 비 오는 곳, 맑은 곳을 한눈에 보여주고 — 거기서 뭘 하면 좋을지까지 알려준다"**

날씨 지도는 정보 도구가 아니라 **코스 전환 엔진**이자 **예약 전환 퍼널**이다.

---

## 2. 타겟 사용자 & 사용 시점

| 사용자 | 시점 | 니즈 | 우리가 주는 것 |
|--------|------|------|---------------|
| **여행 계획 중** (D-7~D-1) | 짐 싸기 전 | "내 여행일 날씨 어때?" | 3일 예보 + AI 코스 자동 조절 |
| **여행 중** (D-day) | 아침에 일어나서 | "오늘 어디 가면 돼?" | 실시간 권역별 날씨 + 맑은 곳 추천 |
| **여행 중** (D-day) | 갑자기 비 올 때 | "비 피할 곳 어디?" | 현위치 기준 실내 장소 즉시 추천 |
| **액티비티 예약자** | 예약 후 | "서핑 가능해?" | 파고/풍속 알림 → 취소 or 대체 연결 |

**핵심 인사이트**: 날씨를 보는 순간 = 의사결정하는 순간 = 예약으로 전환 가능한 순간

## 3. 기능 구조 (3단계)

### Phase 1: 스마트 날씨 대시보드 (현재 → 개선, 1주)

**지금 있는 것**: 10개 지역 기온/강수 카드
**바꿀 것**:

```
┌─────────────────────────────────────────────────────┐
│  🌤️ 제주 실시간 날씨           04.10(목) 14:32 기준  │
├─────────────────────────────────────────────────────┤
│                                                      │
│  [제주 지도 위에 날씨 핀]                              │
│                                                      │
│   한림 ☀️18°   제주시 ☀️20°   함덕 🌧️17°            │
│                                                      │
│   애월 ☀️21°   한라산 ⛅10°   성산 🌧️16°            │
│                                                      │
│   중문 🌤️17°  서귀포 🌧️15°  표선 🌧️16°            │
│                                                      │
├─────────────────────────────────────────────────────┤
│  🟢 지금 맑은 곳: 서쪽 (애월·한림·한경)                │
│  → 추천: 협재해변, 애월카페거리, 새별오름              │
│                                    [코스 만들기 →]   │
├─────────────────────────────────────────────────────┤
│  🔴 지금 비 오는 곳: 동쪽 (성산·구좌·표선)             │
│  → 추천: 아쿠아플라넷, 만장굴, 카페투어               │
│                                    [실내 코스 보기 →] │
└─────────────────────────────────────────────────────┘
```

**핵심 추가 요소**:
- "지금 맑은 곳 / 비 오는 곳" 요약 → 즉각 행동 유도
- 각 요약에 **추천 장소 + 코스 메이커 연결 CTA**
- 카드 클릭 시 해당 권역의 장소 목록 (지도 플랫폼 연결)

### Phase 2: 날씨 연동 코스 전환 (AI 코스 메이커 통합, 2주)

**시나리오**: 여행자가 "2박3일 동쪽 코스"를 만들었는데, 여행 당일 동쪽에 비가 온다.

```
┌──────────────────────────────────────────────┐
│  ⚠️ 코스 날씨 알림                            │
│                                               │
│  내일(4/11) 오전 성산·구좌 지역 비 예보        │
│  강수확률 80% · 예상강수량 15mm                 │
│                                               │
│  현재 코스 Day 2:                              │
│  ❌ 성산일출봉 (야외)                           │
│  ❌ 월정리해변 (야외)                           │
│  ✅ 만장굴 (실내, 유지)                         │
│                                               │
│  AI 추천 대안:                                 │
│  ☀️ 서쪽은 맑음 예보 → 서쪽으로 코스 전환?      │
│                                               │
│  [서쪽 코스로 전환]  [실내 코스로 변경]  [유지]  │
└──────────────────────────────────────────────┘
```

**연결 포인트**:
- AI 코스 메이커에서 생성한 코스에 날씨 자동 모니터링
- 비 예보 시 대안 코스 자동 생성
- "코스 전환" 시 해당 지역 렌터카/숙소/액티비티 예약 유도

## 5. 사용처 (어디서 쓰이는가)

### 5.1 제주패스 앱/웹 메인
- 메인 화면 상단에 "지금 제주 날씨" 위젯
- 탭하면 전체 날씨 지도로 진입
- 비 오는 날엔 자동으로 "비 오는 날 추천" 배너 노출

### 5.2 AI 여행 플래너 연동
- 일정 생성 시 여행일 날씨 예보 자동 반영
- "비 오는 날은 실내 중심으로 구성했어요" 안내

### 5.3 AI 코스 메이커 연동
- 코스 결과에 시간대별 날씨 표시
- 날씨 변경 시 코스 자동 재구성 제안

### 5.4 렌터카 예약 페이지
- 예약 날짜의 날씨 예보 표시
- 악천후 예보 시 "SUV/4WD 추천" 안내
- 태풍/폭설 시 무료 취소/변경 안내

### 5.5 제주패스 SNS/블로그
- "이번 주 제주 날씨" 자동 생성 콘텐츠
- "비 오는 날 가볼 만한 곳 TOP 10" 날씨 연동 콘텐츠
- SEO 유입 → 날씨 지도 → 예약 전환

### 5.6 파트너 업체 대시보드
- 날씨별 예약 트렌드 분석
- "내일 비 예보 → 실내 시설 예약 몰림 예상" 알림
- 날씨 연동 자동 할인 설정 (비 오면 20% 할인 자동 적용)

## 8. 핵심 원칙

1. **날씨는 도구가 아니라 트리거** — 정보를 보여주는 게 아니라 행동을 유발해야 한다
2. **나쁜 날씨 = 최고의 기회** — 비 오면 실내 추천, 더우면 물놀이 추천, 추우면 온천 추천
3. **모든 날씨 화면에 CTA** — 날씨만 보고 나가면 실패, 반드시 다음 행동(코스/예약)으로 연결
4. **다른 프로젝트의 인프라** — 단독 서비스가 아니라 여행 플래너·코스 메이커·지도의 공통 레이어
