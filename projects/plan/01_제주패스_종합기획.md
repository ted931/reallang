### 1.2 매출 — AI 여행 플래너

**개요**: 사용자가 "3박4일 / 아이 둘 / 70만원"처럼 자연어로 입력하면, AI가 숙소·렌터카·식당·액티비티를 패키지로 구성해 즉시 예약까지 연결하는 원스톱 플래너.

| 항목 | 상세 |
|------|------|
| 기술 스택 | GPT-4o 기반 RAG + 제주패스 자체 인벤토리 DB |
| 수익 모델 | 플래너 경유 예약 수수료 12-18% (기존 단순예약 대비 +5%p) |
| 투자 비용 | 초기 개발 1.5억, 월 운영 800만 (API비 포함) |
| KPI | 플래너 전환율 > 8%, 패키지 객단가 > 48만원 |
| 타임라인 | MVP 8주 → 베타 4주 → 정식 런칭 |

**예상 퍼널**:

```
월 방문자 320,000
  → 플래너 진입 64,000 (20%)
    → 플랜 생성 완료 32,000 (50%)
      → 예약 전환 2,560 (8%)
        → 객단가 48만원
          = 월 매출 12.3억 (수수료 기준 1.8억)
```

### 1.3 매출 — 동적 가격 엔진

**개요**: 렌터카 재고·날씨·항공편·축제 일정 등 외부 변수를 실시간 반영해, 차량별 가격을 자동 조정하는 엔진.

| 변수 | 가중치 | 데이터소스 |
|------|--------|-----------|
| 잔여 재고율 | 0.35 | 자체 PMS |
| 항공편 예약 증감 | 0.20 | 한국공항공사 API |
| 날씨 예보 | 0.15 | 기상청 단기예보 |
| 축제/행사 | 0.15 | 제주관광공사 캘린더 |
| 경쟁사 가격 | 0.10 | 크롤링 (롯데/SK 등) |
| 요일/시즌 | 0.05 | 자체 히스토리 |

**기대 효과**:
- 비수기 가동률: 52% → 64% (+12%p)
- 성수기 평균 단가: +18% (수요 대비 과소 책정 해소)
- 연간 추가 매출: 약 5.2억원

### 1.4 매출 — 통합 멤버십 프로그램

**개요**: 렌터카·숙소·액티비티·카페를 하나의 포인트 체계로 묶는 "제주패스 포인트" 도입.

### 1.5 직원 — ERP 시스템 & 성과관리

> 상세 설계는 [3장](#3-직원관리-erp-설계) 참조

**현재 문제점**:
- 근태관리: 수기 엑셀 → 월말 정산에 3일 소요
- 시프트 배정: 카카오톡 수동 조율 → 월 평균 8건 누락
- 성과 평가: 연 1회 정성평가 → 피드백 사이클 12개월
- 급여 계산: 수당 종류 11개, 수작업 오류 월 2-3건

**목표 상태**:
- 자동 근태 + GPS 출퇴근 체크
- AI 기반 시프트 최적화 (수요 예측 연동)
- 월간 OKR + 실시간 대시보드
- 급여 자동 산출 → 검증 → 이체

#### 3.3.2 AI 시프트 최적화

**입력 변수**:
- 익일 예약 건수 (수요예측 모델 연동)
- 직원별 선호 시간대 (가중치 0.3)
- 연속 근무일수 (5일 초과 시 강제 휴무)
- 스킬 매칭 (차량 인수 가능 여부 등)

**최적화 목표**:
- 제약 1: 최소인원 충족
- 제약 2: 노동법 준수 (주 52시간)
- 제약 3: 연속 야간 2일 이하
- 목적함수: 인건비 최소화 + 직원 만족도 최대화 (Pareto 최적)

**알고리즘**: Constraint Programming (Google OR-Tools) + 직원 선호도 소프트 제약

#### 3.3.3 시프트 운영 플로우

```
매주 수요일 14:00
  → AI가 차주 시프트 초안 생성 (5분)
    → 팀장 검토/수정 (수요일 내)
      → 목요일 09:00 전직원 알림 (카카오톡 + 앱)
        → 금요일 18:00까지 변경 요청 가능
          → 월요일 확정 → 근태 시스템 자동 반영
```

## 5. 무료 홍보 플랫폼 기획

### 5.1 핵심 전략 요약

> **한 줄 정의**: 제주도 자영업자에게 **무료 홍보 페이지**를 제공하고, 이 페이지들의 SEO 파워를 모아 **제주패스 도메인 권위**를 높인 뒤, **카페패스** 구독 상품으로 수익화한다.

```
[자영업자 무료 가입] → [가게 홍보 페이지 자동 생성]
    → [SEO 최적화 콘텐츠 수천 페이지]
        → [네이버/구글 상위 노출]
            → [여행객 유입 → 렌터카/카페패스 전환]
```

### 5.2 사업자 인증 체계

| 5. 관리자 승인 | 중복/부적절 필터링 | 24시간 이내 | 자동 90% + 수동 10% |

### 6.4 리스크 3가지

#### 리스크 1: SEO 트래픽이 기대에 못 미칠 확률 — 60%

| 시나리오 | 확률 | 월 유입 | 대응 |
|---------|------|--------|------|
| 낙관 | 20% | 100,000+ | 계획대로 진행 |
| 기본 | 40% | 30,000-50,000 | 블로그/인플루언서 보완 |
| 비관 | 40% | 10,000 이하 | SEO 포기, 바이럴/오프라인 집중 |

#### 리스크 2: 사장님 이탈률이 95%를 초과할 확률 — 35%

사장님 지속률이 5% 미만으로 떨어지면 콘텐츠 생산이 멈추고 SEO 효과도 소멸.

**완화 전략**: 가입 즉시 자동 페이지 생성 → 사장님이 이탈해도 페이지는 유지 → SEO 효과 지속

#### 리스크 3: 카페패스 수익화 실패 확률 — 30%

여행객이 개별 결제를 선호하거나, 제휴 가게 수가 임계치(100개) 미달 시 상품 매력도 부족.

**완화 전략**: 최소 100개 제휴 확보 후 런칭 / 가격을 "1잔 값"(5,900원)으로 설정하여 허들 최소화

### 6.5 4주 검증 스프린트

| 주차 | 목표 | 활동 | 성공 기준 |
|------|------|------|----------|

### 7.2 사장님 바이럴 — 3대 무기

#### 7.2.1 카카오톡 마케팅 대행

**사장님의 가장 큰 고민**: "홍보해야 하는 건 아는데 시간이 없다"

| 서비스 | 내용 | 사장님 액션 | 비용 |
|--------|------|-----------|------|
| 알림톡 자동 발송 | 방문 고객에게 감사 메시지 + 쿠폰 | 없음 (자동) | 건당 15원 (제주패스 부담) |
| 카카오 채널 대행 | 주 2회 포스팅 자동 생성/발행 | 승인 1탭 | 무료 |
| 친구 초대 캠페인 | "사장님 가게 카카오 채널 친구 1,000명 만들기" | 오프라인 QR 비치 | 무료 |

**핵심 가치**: "사장님은 가게만 잘 운영하세요. 홍보는 제주패스가 하겠습니다."

**바이럴 메커니즘**: 사장님이 다른 사장님에게 추천 → "나 여기 등록했더니 카톡 홍보 다 해줘" → 입소문

#### 7.2.2 AI 가게 사진 서비스

| 서비스 | 설명 | 품질 기준 |
|--------|------|----------|
| AI 보정 | 사장님 촬영 사진 → 전문가급 보정 | 밝기/색감/구도 자동 최적화 |
| 메뉴 촬영 가이드 | "이 각도에서 이렇게 찍으세요" AI 가이드 | 실시간 카메라 오버레이 |
| 배경 생성 | 식탁/감성 배경 자동 합성 | FID score < 50 |
| 카드뉴스 | 사진 + 가게 정보 → 인스타 카드뉴스 | 1080x1080, 4장 세트 |

**사장님 반응 시나리오**:
- "이걸 무료로 해준다고? 당장 해야지" → 가입 동기
- "내 가게 사진이 이렇게 나오네!" → SNS 자랑 → 다른 사장님 유입

#### 7.2.3 매출 직결 피처

사장님이 이탈하지 않는 유일한 이유는 **실제 매출 증가를 체감**할 때.

| 피처 | 매출 연결 방식 | 측정 |
|------|--------------|------|
| 카페패스 주문 | 여행객이 카페패스로 결제 → 실시간 매출 알림 | 일별 카페패스 매출 |
| 예약 연동 | 제주패스 앱에서 예약 → 실시간 예약 알림 | 월별 예약 건수 |
| 쿠폰 유입 추적 | 제주패스 쿠폰으로 방문 → "제주패스에서 N명 왔어요" | 쿠폰 사용 건수 |
| 검색 노출 리포트 | "이번 주 '애월 카페' 검색에 3위 노출" | 키워드 순위 |
