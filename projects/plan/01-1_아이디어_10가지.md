## 1. 아이디어 10가지 요약

제주패스의 현재 비즈니스(렌터카 예약 + 여행상품 연결)를 기반으로, **매출·직원·고객·운영·데이터** 5개 축에서 도출한 핵심 아이디어 10가지를 정리한다.

### 1.1 전체 맵

| # | 축 | 아이디어 | 핵심 지표 | 예상 임팩트 |
|---|-----|---------|----------|------------|
| 1 | 매출 | AI 여행 플래너 | 객단가 +35% | 연 매출 +8억 |
| 2 | 매출 | 동적 가격 엔진 | 가동률 +12%p | 연 매출 +5억 |
| 3 | 매출 | 통합 멤버십 프로그램 | 재방문율 40%→58% | LTV +220만 |
| 4 | 직원 | 직원관리 ERP | 행정시간 -60% | 연 1.2억 절감 |
| 5 | 직원 | 성과관리 시스템 | 이직률 25%→15% | 채용비 -4,000만 |
| 6 | 고객 | AI CS 챗봇 | 응답시간 8분→15초 | CS인력 -3명 |

| 10 | 운영 | 파트너 포털 | 정산 오류 -90% | 파트너 NPS +30p |

### 1.2 매출 — AI 여행 플래너

**개요**: 사용자가 "3박4일 / 아이 둘 / 70만원"처럼 자연어로 입력하면, AI가 숙소·렌터카·식당·액티비티를 패키지로 구성해 즉시 예약까지 연결하는 원스톱 플래너.

| 항목 | 상세 |
|------|------|
| 기술 스택 | GPT-4o 기반 RAG + 제주패스 자체 인벤토리 DB |
| 수익 모델 | 플래너 경유 예약 수수료 12-18% (기존 단순예약 대비 +5%p) |
| 투자 비용 | 초기 개발 1.5억, 월 운영 800만 (API비 포함) |
| KPI | 플래너 전환율 > 8%, 패키지 객단가 > 48만원 |
| 타임라인 | MVP 8주 → 베타 4주 → 정식 런칭 |

**예상 퍼널**:

```
월 방문자 320,000
  → 플래너 진입 64,000 (20%)
    → 플랜 생성 완료 32,000 (50%)
      → 예약 전환 2,560 (8%)
        → 객단가 48만원
          = 월 매출 12.3억 (수수료 기준 1.8억)
```

### 1.3 매출 — 동적 가격 엔진

**개요**: 렌터카 재고·날씨·항공편·축제 일정 등 외부 변수를 실시간 반영해, 차량별 가격을 자동 조정하는 엔진.

| 변수 | 가중치 | 데이터소스 |
|------|--------|-----------|
| 잔여 재고율 | 0.35 | 자체 PMS |
| 항공편 예약 증감 | 0.20 | 한국공항공사 API |
| 날씨 예보 | 0.15 | 기상청 단기예보 |
| 축제/행사 | 0.15 | 제주관광공사 캘린더 |
| 경쟁사 가격 | 0.10 | 크롤링 (롯데/SK 등) |
| 요일/시즌 | 0.05 | 자체 히스토리 |

**기대 효과**:
- 비수기 가동률: 52% → 64% (+12%p)
- 성수기 평균 단가: +18% (수요 대비 과소 책정 해소)
- 연간 추가 매출: 약 5.2억원

### 1.4 매출 — 통합 멤버십 프로그램

**개요**: 렌터카·숙소·액티비티·카페를 하나의 포인트 체계로 묶는 "제주패스 포인트" 도입.

| 등급 | 조건 (연간) | 혜택 | 예상 비율 |
|------|------------|------|----------|
| 블루 | 가입 즉시 | 1% 적립, 생일 5,000P | 60% |
| 그린 | 50만원 이상 | 3% 적립, 무료 차량 업그레이드 1회 | 25% |
| 오렌지 | 150만원 이상 | 5% 적립, 공항 픽업, 우선 배차 | 12% |
| 블랙 | 300만원 이상 | 7% 적립, 전용 컨시어지, VIP 라운지 | 3% |

**LTV 시뮬레이션**:
- 비멤버 평균 LTV: 42만원 (1.2회 방문)
- 멤버 평균 LTV: 264만원 (3.8회 방문) → **+220만원**
- 멤버십 운영비: 연 1.8억 (포인트 소진 + 시스템)
- ROI: 14.7x (1년 기준)

### 1.5 직원 — ERP 시스템 & 성과관리

> 상세 설계는 [3장](#3-직원관리-erp-설계) 참조

**현재 문제점**:
- 근태관리: 수기 엑셀 → 월말 정산에 3일 소요
- 시프트 배정: 카카오톡 수동 조율 → 월 평균 8건 누락
- 성과 평가: 연 1회 정성평가 → 피드백 사이클 12개월
- 급여 계산: 수당 종류 11개, 수작업 오류 월 2-3건

**목표 상태**:
- 자동 근태 + GPS 출퇴근 체크
- AI 기반 시프트 최적화 (수요 예측 연동)
- 월간 OKR + 실시간 대시보드
- 급여 자동 산출 → 검증 → 이체

### 1.7 고객 — 실시간 여행 허브

**개요**: 제주도 실시간 정보(교통·날씨·혼잡도·맛집 대기시간)를 한 화면에 모아, 여행 중 "지금 뭐하지?"에 답하는 허브.

| 데이터 | 소스 | 갱신주기 |
|--------|------|---------|
| 도로 교통 | 제주특별자치도 TOPIS | 5분 |
| 관광지 혼잡도 | SKT 유동인구 API | 15분 |
| 날씨/풍속 | 기상청 초단기 | 30분 |
| 맛집 대기 | 자체 파트너 POS 연동 | 실시간 |
| 액티비티 잔여석 | 파트너 API | 실시간 |

**수익화**: 허브 내 네이티브 광고 (CPM 12,000원) + 즉석 예약 수수료 → 연 2.1억

### 1.8 고객 — 리뷰 엔진

**현재**: 리뷰 작성률 4.2%, 평균 리뷰 길이 18자 ("좋았어요" 수준)

**개선 설계**:

| 장치 | 설명 | 기대 효과 |
|------|------|----------|
| 포토 리뷰 인센티브 | 사진 3장 이상 → 3,000P | 리뷰율 +8%p |
| AI 리뷰 초안 | 예약 이력 기반 "이런 점이 좋았어요" 프리필 | 완료율 +40% |
| 타이밍 최적화 | 반납 후 2시간, 만족도 피크 시점 푸시 | 응답률 2.3x |
| 리뷰 게이미피케이션 | "제주 리뷰어" 배지 + 랭킹 | 재작성률 +25% |
| 사장님 답글 알림 | 업주가 답글 달면 리뷰어에게 알림 | 참여 루프 강화 |

**목표**: 리뷰율 4.2% → 18%, 평균 길이 18자 → 85자, 포토 비율 12% → 55%
