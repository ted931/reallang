# 제주패스 파트너 광고 프로그램 — 서비스 타당성 종합 리포트

**작성일**: 2026-04-28  
**분석팀**: 업체(B2B) · 사용자(B2C) · 수요/공급 · 외부 리스크 · 독립 사이트 vs 애드온 · jejupass.com 수익화 (6개 팀 병렬 분석)

---

## 최종 판정

> **조건부 타당 — 즉시 시작하되, 순서를 지켜야 한다.**

광고/파트너 수익 모델 자체는 시장 타이밍, 포지셔닝 모두 타당하다. 단, 지금 당장 업체 영업을 나가면 실패한다. **트래픽 확보 → 검증 인프라 → 유료 전환**의 순서를 지켜야 한다.

---

## 1. 업체(B2B) 관점 — 팀 1 요약

### 결론: 즉시 지불 의향 낮음 → 트래픽 증명 후 중간~높음

**가장 매력적인 타겟 업체 (우선순위 순):**

| 순위 | 업종 | 이유 | 월 광고비 적정선 |
|------|------|------|----------------|
| 1 | 게스트하우스 | 빈 방 1개만 채워도 ROI, 파티와 직결 | 10~30만원 |
| 2 | 액티비티 업체 | 그룹 전환, 객단가 높음, 글로벌 대안보다 저렴 | 15~50만원 |
| 3 | 렌탈 업체 | 파티 단체 방문 시 복수 전환 | 5~15만원 |
| 4 | 카페 | 객단가 낮아 쿠폰 부담 큼, 초기 제외 권장 | 3만원 이하만 가능 |

**권장 요금 체계:**

| 플랜 | 월 금액 | 제공 |
|------|---------|------|
| 프리 (무료) | 0원 | Basic 카드 노출만 |
| 베이직 | 3만원 | Basic + 할인코드 + 통계 |
| 피처드 | 10만원 | Featured 카드 (풀 와이드) |
| 프리미엄 | 30만원+ | Featured + 홈 배너 + 전용 캠페인 |

**가장 큰 온보딩 장벽:** 쿠폰 검증 시스템 없음. 업체는 "손님이 코드 들고 오면 어떻게 확인하냐"를 먼저 물어본다. 이게 해결되기 전까지는 어떤 업체도 유료 계약을 하지 않는다.

## 2. 사용자(B2C) 관점 — 팀 2 요약

### 결론: 종합 61/100점 → 3가지 수정 시 75점+ 가능

**현재 UX의 강점:**
- 파티 카테고리 기반 맥락 광고 → "광고처럼 안 느껴짐"
- "파티 참가자 혜택" 레이블이 사용자 이익 관점으로 재프레이밍
- freebie 오퍼(감귤 음료 무료 등)의 심리적 장벽 낮음

**즉시 수정해야 할 3가지:**

1. **외부 지도 링크 추가** (`location` → 카카오맵 검색 링크)  
   → 업체 신뢰도가 가장 빠르게 상승하는 방법

2. **할인 절대금액 표시** ("15% 할인" → "₩12,000 절약")  
   → 카페패스 섹션의 절약액 패턴을 그대로 적용

3. **사용 조건 명확화** ("파티 참가 확인서 지참" → QR/스크린샷 인증 방법 명시)  
   → 불명확한 조건이 클릭 포기의 가장 큰 원인

**추가 개선:**
- `targetCategories: ["all"]` 오퍼는 관련 카테고리 오퍼가 2개 미만일 때만 보충 노출
- 참여 완료(`paymentStep === "done"`) 화면에서 파트너 오퍼 재노출 (전환율 최고 시점)

## 3. 수요/공급 시장 관점 — 팀 3 요약

### 결론: 시장은 충분히 크나, 파티 월 300건 전까지는 광고 단독 수익화 불가

**시장 규모 추정:**

| 구분 | 연간 규모 |
|------|---------|
| TAM (제주 관광 업체 광고 전체) | 약 29억원 |
| SAM (액티비티 연관 업체) | 약 9억원 |
| SOM (3년 내 현실적 목표) | 약 3.84억원 (파트너 400개 × 월 8만원) |

**Cold Start 해결책 — 가장 현실적인 방법:**
- jeju-car 렌터카 예약 완료 화면에서 파티 추천 카드 연결
- 월 500건 예약 × 5% 전환 = 월 25명 신규 파티 참가자 (광고비 0원)
- 이 교차 트래픽 전략이 유일한 탈출구

**경쟁 현황:** 직접 경쟁자 없음. "제주 여행자 소셜 파티 + 렌터카 연계" 포지션을 가진 서비스가 국내에 없다. 프립이 제주 강화할 경우 유일한 위협.

**계절성 대비:** 성수기(7~8월) + 준성수기(4~6, 9~10월)에 수익 70% 집중. 비수기에는 한달살이 파티, 로컬 커뮤니티 파티로 방어.

## 4. 외부 리스크 관점 — 팀 4 요약

### 결론: Top 3 리스크 — 설계 단계에서 반드시 반영해야 함

**리스크 매트릭스:**

| 순위 | 리스크 | 위험도 | 완화 전략 |
|------|--------|--------|-----------|
| 1 | 파티 안전사고 법적 책임 | ★★★★★ | 고위험 카테고리 안전 체크리스트 + 단체보험 파트너십 |
| 2 | 개인정보법 행동 타겟팅 광고 동의 위반 | ★★★★ | 콘텐츠 기반(카테고리) 광고로 설계 (개인 행동 프로파일 불사용) |
| 3 | 오프라인 쿠폰 검증 인프라 부재 | ★★★★ | QR 스캔 기반 업체용 웹앱 우선 구축 |

**법적 필수 조항:**
- **표시광고법**: "협찬", "광고" 표시 UI에 반드시 명시 (기만광고 제재 방지)
- **개인정보보호법 제37조의2**: 행동 기반 광고 매칭 시 별도 동의 필요 → 카테고리 기반으로 회피 가능
- **전자상거래법**: 쿠폰 결제 대행 추가 시 통신판매중개업 등록 필요

**긍정적 발견:** 경기 침체는 쿠폰 모델에 오히려 유리. 제주 오버투어리즘 규제는 분산 관광 수요를 늘려 플랫폼 가치 상승 가능.

## 5. 독립 사이트 vs 애드온 관점 — 팀 5 요약

### 결론: 하이브리드 단계적 접근 권장

> **"jeju-party 안에서 검증하고, jeju-biz로 이전한다"**

**핵심 판단 기준 3가지:**
1. **트래픽이 먼저다** — B2B 포털은 트래픽의 결과물, 월 1,000명 이전에 독립 사이트 투자는 선행 조건 역전
2. **여행자 UX가 파트너 신뢰의 원천** — 사용자 앱이 신뢰받아야 업체가 광고비를 낸다
3. **jeju-biz(포트 3022)가 이미 존재** — 새 사이트 불필요, `/partner` 라우트 하나면 충분

**단계별 로드맵:**

| 단계 | 기간 | 핵심 행동 |
|------|------|-----------|
| Phase 0 | 지금~2주 | dummy-partners → DB 마이그레이션, 파트너 3개 무료 유치, 클릭 이벤트 로깅 |
| Phase 1 | 1개월 | 참여 완료 화면 파트너 카드 재노출, 쿠폰 QR 검증 웹앱 |
| Phase 2 | 2~3개월 | jeju-biz `/partner` 셀프서브 대시보드 (기존 컴포넌트 재활용) |
| Phase 3 | 6개월+ | 파트너 30개, 월 정산 300만원 달성 후 독립 포털 전환 고려 |

## 6. jejupass.com 수익화 관점 — 팀 6 요약

### 결론: 코드 변경 최소로 월 1,100~2,100만원 신규 수익 즉시 가능

**이미 있는 인프라:**
- B2B 화이트라벨 API: 인터파크, 트리플, KB Pay에 이미 렌터카 API 제공 중
- GA + GTM: 광고 전환 추적 인프라 완비
- 렌터카 81개 업체 18,270대 + 카페패스 200개 카페 → 광고 인벤토리로 즉시 전환 가능

**Phase 0 수익화 (4주 내, 구현 최소):**

| 전략 | 예상 월 수익 | 구현 난이도 |
|------|------------|-----------|
| 여행 보험 CPA (예약 완료 화면 삽입) | 500~1,000만원 | FE 1인 1주 |
| 렌터카 업체 스폰서드 리스팅 (상위 5~10개사) | 300~500만원 | UI 수정만 |
| 카페패스 파트너 카페 스폰서 노출 | 300~600만원 | DB 필드 추가 |
| **합계** | **1,100~2,100만원/월** | |

**업계 벤치마크:**
- 야놀자: 광고 수익이 전체 14.2% (연 약 201억원)
- 마이리얼트립: 광고/플랫폼 기타 매출 YoY +390%, 창사 첫 흑자 달성의 결정적 레버

**16개 서비스 통합 광고 네트워크:**
jeju-party, jeju-car, smart-fuel, weather-drive 등 16개 서비스를 단일 광고 네트워크로 묶으면 인벤토리 급증. 광고주가 16개 채널에 한 번에 집행하는 **통합 제주 광고 플랫폼**으로 발전 가능.

## 종합 판단 및 실행 우선순위

### 하면 안 되는 것

| ❌ | 이유 |
|----|------|
| 지금 당장 업체 유료 영업 나가기 | 트래픽 없으면 ROI 설명 불가, 첫 인상 망침 |
| 독립 B2B 광고 사이트 신규 개발 | jeju-biz 이미 있음, 낭비 |
| `targetCategories: ["all"]` 오퍼 남발 | 맥락 파괴, 사용자/업체 신뢰 하락 |
| 정적 할인코드 그대로 사용 (검증 없이) | 어뷰징, 업체 신뢰 붕괴 |

### 즉시 해야 하는 것

**이번 주 (코드 작업):**
1. 파트너 오퍼 카드에 카카오맵 링크 추가 (`location` 필드 활용)
2. 할인 절대금액 계산 표시 ("15% 할인" → "₩12,000 절약")
3. 쿠폰 복사 이벤트 서버 로깅 (업체 영업 시 ROI 데이터로 활용)
4. 참여 완료 화면에 파트너 오퍼 재노출 (전환율 최고 시점)

**이번 달 (비즈 작업):**
5. 게스트하우스 3개, 액티비티 업체 2개를 **무료 베타 파트너**로 계약
6. jeju-car 예약 완료 → jeju-party 파티 추천 연결 (Cold Start 해결)

**2~3개월 내 (인프라):**
7. QR 기반 쿠폰 검증 업체용 웹앱 구축
8. jeju-biz `/partner` 셀프서브 대시보드 (노출수 + 코드 복사수)
9. jejupass.com 예약 완료 화면 여행보험 CPA 삽입

### 수익 시나리오

| 시나리오 | 조건 | 월 수익 |
|----------|------|---------|
| 보수적 | 파트너 30개 × 5만원 | 150만원 |
| 기본 | 파티 MAU 5,000, 파트너 100개 × 10만원 | 1,000만원 |
| 낙관적 | jejupass 전체 광고 인벤토리 통합 | 2,000~5,000만원 |

## 한 줄 결론

**사업성은 있다. 다만 "광고 수익"이 아니라 "트래픽 성장"이 선행 지표다. 지금 당장 해야 할 일은 광고 영업이 아니라 jeju-car → jeju-party 연결과 코드 3줄 수정이다.**
