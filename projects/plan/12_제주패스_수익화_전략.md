## 1. 플랫폼 현황 분석

### 확인된 서비스 구성

크롤링 결과, 제주패스는 단순 렌터카 예약 중개를 넘어 다음 영역을 커버하는 멀티버티컬 플랫폼으로 성장 중이다.

| 서비스 | 설명 | 규모 |
|--------|------|------|
| 렌터카 가격비교 | 국내(제주) + 해외(일본 등) | 81개 업체, 450종, 18,270대 |
| 카페패스 | 제주 인기 카페 200여 곳 음료 무료 | 무제한 이용권 형태 |
| 숙박 | 전국 호텔·리조트·펜션·카라반 | 전국 확장 중 |
| 트립/액티비티 | 제주·오키나와 오리지널 체험 상품 | 해외 확장 중 |

### 기술적으로 이미 구현된 수익화 인프라

코드에서 확인된 것들:

- **화이트라벨 B2B 채널**: interparkos(인터파크), tripleos(트리플), KBPAY(KB국민카드 페이) 등 이미 파트너 채널이 존재
- **Google Analytics + GTM**: 사용자 행동 전체 추적 중 (전환 최적화 데이터 확보 가능)
- **환경 분리**: dev/prod 분리, 팝업 노출 제어 로직 — 광고 A/B 테스트 인프라 준비 완료

### 플랫폼 포지셔닝

```
         가격비교 플랫폼
              ↓
         렌터카 예약 중개
              ↓
    (현재) 카페패스 + 숙박 + 액티비티
              ↓
    (목표) 제주 여행 슈퍼앱
```

## 2. 수익화 모델 카탈로그

### 2-1. 광고 수익

#### A. 렌터카 업체 검색 광고 (스폰서드 리스팅)
- **구조**: 가격 비교 결과 최상단에 "협찬" 태그 카드 노출
- **과금 방식**: CPC (클릭당 500~2,000원) 또는 CPL (견적 요청당 5,000~15,000원)
- **수익 추정**: MAU 30만 가정, 전환 검색 10%, CTR 5%, CPC 1,000원 → 월 **150만원**
- 81개 업체 중 상위 10개 업체만 참여해도 안정적 인벤토리
- **레퍼런스**: 네이버 항공권/숙박 검색 광고, 다이렉트 보험 비교 사이트

#### B. 카페/맛집 네이티브 광고
- **구조**: 카페패스 지도/리스트 화면에 "오늘의 추천 카페" 스폰서드 카드 삽입
- **과금 방식**: 카페당 월 고정 30~100만원 (노출 보장형)
- **수익 추정**: 제휴 카페 200곳 중 스폰서 20곳 × 50만원 = 월 **1,000만원**
- UX 영향 최소화: 일반 카드와 동일한 디자인, "파트너" 뱃지만 추가

#### C. 숙박/액티비티 디스플레이 배너
- **위치**: 렌터카 예약 완료 후 "다음 여행은?" CTA 영역, 메인 히어로 하단 슬롯
- **형태**: 카드형 네이티브 배너 (300×250, 728×90 회피 — 여행 감성 카드 형태)
- **과금 방식**: CPM 3,000~5,000원 (여행 카테고리 프리미엄)
- **수익 추정**: 월간 노출 500만 회 기준 → 월 **150~250만원**

#### D. 제주 관광지/체험 업체 스폰서십
- 협재해수욕장 주변 카페, 성산일출봉 주변 액티비티 업체 등 지역 밀착형
- 지도 뷰에서 핀 강조 표시 + 상세 페이지 최상단 노출
- 소상공인 대상 월 10~30만원 소액 패키지로 진입 장벽 최소화

### 2-2. 파트너십 수익

#### A. 화이트라벨 B2B API (이미 일부 운영 중)
- 인터파크, 트리플, KB Pay에 이미 제공 중인 화이트라벨을 확장
- **신규 타깃**: 제주항공(파트너십 흔적 확인됨), 카카오T, 네이버 여행, 카드사 혜택 앱
- **수익 구조**: 연간 API 사용료 2,000~5,000만원 + 예약 건당 수수료 1~3%
- **수익 추정**: 파트너 5개 추가 × 연 3,000만원 = 연 **1.5억원**

#### B. 렌터카 업체 부스팅 수수료 차등화
- 현재: 모든 업체 동일 노출
- 전환: 기본 수수료(예: 10%) + 프리미엄 노출 수수료(+2~3%) 이원화
- 업체 입장에서 추가 수수료 = 마케팅비로 인식 → 수용 저항 낮음

#### C. 여행 보험 CPA 파트너십
- 렌터카 예약 시 "여행자 보험 추가" 옵션 삽입
- 삼성화재/현대해상/DB손보 등 보험사와 CPA 계약
- 보험 1건당 3,000~8,000원 수수료
- **수익 추정**: 월 예약 5만 건 중 10% 전환 = 5,000건 × 5,000원 = 월 **2,500만원**
- 클룩, 야놀자 모두 보험 번들 CPA로 의미있는 수익 창출 중

#### D. 주유소/EV 충전 제휴
- smart-fuel 서비스 연동: 특정 주유소 "파트너 주유소" 지정
- 해당 주유소 방문 시 적립 포인트 + 제주패스 수수료
- 제주에만 약 130개 주유소 → 20~30개 파트너 확보 가능
- 월 30만원 × 25개 = 월 **750만원**

#### E. 카드사 제휴 할인 프로모션
- KB Pay 이미 연동 → 국민카드 전용 할인 이벤트
- 추가: 신한카드, 현대카드, 롯데카드 등 분기별 제휴 프로모션
- 카드사가 마케팅 예산을 지불하는 구조 (플랫폼에 비용 없음)
- 건당 수수료 또는 고정 스폰서 비용 수취

### 2-3. 프리미엄 기능 / 멤버십

#### A. 제주마일 (로열티 프로그램) — 기획서 기존 항목
- 렌터카 이용 거리/횟수/리뷰 마일리지 → 등급별 혜택
- **무료**: 기본 가격비교 예약
- **실버 (연 1만원)**: 카페패스 월 3잔 무료 쿠폰, 예약 우선순위
- **골드 (연 3만원)**: 카페패스 무제한, 액티비티 10% 할인, 무료 업그레이드
- **수익 추정**: 가입자 10만 명 중 유료 전환 5% = 5,000명 × 평균 2만원 = 연 **1억원**
- **핵심 효과**: 가격 경쟁 탈피 + 재방문율 상승

#### B. 비즈니스 플랜 (법인/단체 여행)
- 기업 단체 여행, 인센티브 투어 팀 대상
- 대량 예약 할인 + 영수증 통합 발행 + 담당 매니저 배정
- 법인 카드 직결 + 세금계산서 발행
- 월 구독 50~200만원 (기업 규모별)
- **수익 추정**: 가입 법인 100개 × 평균 월 100만원 = 월 **1억원**

#### C. 얼리버드/최저가 알림 유료 구독
- 성수기 렌터카 "최저가 알림" 선착순 예약 기능
- 월 3,000~5,000원 구독
- 성수기 제주 렌터카 공급 부족 문제 직접 해소 → 가치 명확

## 4. 광고 위치 제안 (UX 최소 침해 원칙)

### 원칙
1. **여행 감성 카드 형태** — 일반 배너 금지, 콘텐츠와 동일한 카드 UI
2. **맥락 광고** — 렌터카 결과 → 주유소 광고, 카페패스 → 카페 업체 광고
3. **노출 빈도 제한** — 동일 광고 1일 3회 이상 노출 금지
4. **명확한 레이블** — "파트너", "스폰서" 뱃지 필수 (신뢰도 유지)

### 위치별 제안

```
┌─────────────────────────────────┐
│  헤더 네비게이션                 │  ← 광고 없음 (UX 핵심 영역)
├─────────────────────────────────┤
│  메인 히어로 배너                │  ← 계절 프로모션 스폰서십 가능
│  (렌터카 검색 폼)               │    (예: "KB Pay로 결제하면 5% 할인")
├─────────────────────────────────┤
│  카테고리 탭                     │  ← 광고 없음
│  (렌터카/카페패스/숙박/트립)     │
├─────────────────────────────────┤
│  검색 결과 리스트                │
│  ┌───────────────────────────┐  │  ← 3번째 카드: 스폰서드 리스팅
│  │ [파트너] 업체명 카드       │  │    (일반 카드와 동일 디자인,
│  └───────────────────────────┘  │     "파트너" 배지만 추가)
│  일반 업체 카드 × N             │
├─────────────────────────────────┤
│  예약 완료 화면                  │  ← "여행을 더 풍성하게" 섹션
│                                 │    - 여행 보험 CTA (가장 고전환)
│                                 │    - 카페패스 업셀
│                                 │    - 주유소 파트너 쿠폰
├─────────────────────────────────┤
│  카페패스 지도 화면              │  ← 지도 핀 스폰서 강조
│                                 │    (일반 핀과 색상만 다름)
├─────────────────────────────────┤
│  푸터 상단                      │  ← 파트너 배너 (항공사, 카드사)
│                                 │    300×100 소형, 2~3개
└─────────────────────────────────┘
```

### 절대 광고 금지 영역
- 검색 결과 1, 2번째 카드 (신뢰도 훼손)
- 예약 폼 중간 (전환율 타격)
- 모바일 화면 하단 고정 배너 (한국 모바일 사용자 기피도 높음)
- 카페패스 상세 페이지 내부 (콘텐츠 순도 유지)

| 수익 구성 | 예약 수수료 + **광고/플랫폼 기타 매출 YoY +390%** |
| 마케팅 파트너 | SNS 공유 링크 수익의 50% 지급 |
| 특징 | 광고 수익이 예약 수수료 의존도를 낮추는 핵심 |
| 시사점 | **플랫폼 기타 매출(광고)을 조기에 구축**하면 흑자 전환 레버리지 |

### 5-3. 에어비앤비 체험

| 항목 | 내용 |
|------|------|
| 수수료 | 호스트에게 20% 수수료 (숙박 대비 높음) |
| 큐레이션 가치 | 체험은 품질 보증 비용이 높아 수수료 정당화 |
| 특징 | 에셋 라이트: 호스트가 운영, Airbnb는 마케팅+결제+큐레이션만 |
| 시사점 | **제주 트립 서비스**에 동일 모델 적용: 20% 수수료 + 품질 보증 |

## 7. 16개 제주 서비스 교차 수익화 시너지

팀이 운영 중인 서비스들을 단일 광고 네트워크로 묶으면 인벤토리가 급증한다.

```
제주패스 (메인 허브)
    ├─ jeju-party  → 파티 참가자에게 렌터카 광고
    ├─ jeju-car    → 렌터카 예약 후 카페패스 광고
    ├─ smart-fuel  → 주유소 쿠폰 + 렌터카 업셀
    ├─ weather-drive → 드라이브 코스 + 카페패스 CTA
    ├─ jeju-map    → 지도 내 스폰서 핀 광고
    └─ jeju-biz    → 법인 여행 패키지 광고
```

**교차 수익화 모델**:
1. 공통 사용자 ID로 행동 데이터 통합 → 개인화 광고 CTR 3배
2. 서비스 간 번들 쿠폰 순환 → 이탈 방지 + 재방문 유도
3. 광고주가 16개 서비스에 한 번에 집행 가능한 통합 광고 대시보드

## 9. 실행 시 주의사항

### 수익화 원칙
1. **광고 밀도 제한**: 화면당 광고 최대 1~2개. 야놀자/클룩의 과도한 광고가 UX 불만 원인
2. **콘텐츠와 광고의 명확한 구분**: "파트너", "스폰서" 레이블 필수 — 신뢰 기반 플랫폼 유지
3. **기존 핵심 가치 보호**: "최저가 실시간 비교"가 흔들리면 플랫폼 정체성 붕괴. 스폰서 카드는 정렬 순서에 영향 없이 추가 노출로 처리
4. **개인정보 동의**: 행동 기반 광고 집행 전 개인정보 처리방침 업데이트 필수

### 빠른 실행을 위한 첫 3가지 액션
1. **이번 주**: 렌터카 상위 10개 업체에 스폰서드 리스팅 제안서 발송
2. **다음 주**: 삼성화재/현대해상 보험 API 파트너십 미팅 (예약 완료 화면 CTA)
3. **이번 달**: 카페패스 파트너 카페 20곳에 스폰서 패키지 영업 시작
