# 피싱로그 (FishingLog) — 발표용 기획서

> **대상 청중:** 투자자 · 지역 낚시업체 대표 · 제주관광청 담당자
> **작성일:** 2026년 5월

---

## 1. 서비스 개요

**"제주 낚시꾼과 낚시 업체를 연결하는 올인원 낚시 플랫폼"**

조황 공유부터 예약·주문·커뮤니티까지, 낚시 라이프사이클 전체를 하나의 앱에서 해결한다.

---

## 2. 문제 정의

### 낚시꾼이 겪는 불편
- **정보 파편화** — 조황 정보가 카카오톡 단체방, 블로그, 유튜브에 분산돼 신뢰도가 낮음
- **예약 불편** — 좌대·배편·낚시 숙소를 각각 다른 경로로 예약해야 함
- **채비·미끼 구입** — 출조 전날 마트를 돌거나 배송을 기다려야 함
- **물때·포인트 정보 부재** — 검증된 포인트 지도와 실시간 물때를 한 번에 볼 수 없음
- **합동출조 조율 어려움** — 일행 모집이 단톡·커뮤니티 수기 조율에 의존

### 업체(사장님)가 겪는 불편
- **디지털 노출 없음** — 제주 낚시업체 대부분 SNS·예약 시스템 미보유
- **예약 관리 수동** — 전화·문자로만 예약을 받아 이중예약·노쇼 빈발
- **마케팅 비용 과다** — 플랫폼 없이 자체 홍보 → 단골 외 신규 유입 없음
- **매출 데이터 미활용** — 시즌별 수요 예측·재고 관리를 감에 의존

---

## 3. 솔루션

피싱로그는 **낚시 라이프사이클 전 단계를 수직 통합**한 플랫폼이다.

| 단계 | 기존 방식 | 피싱로그 |
|------|-----------|----------|
| 정보 수집 | 카톡·블로그 산재 | 조황 피드 + 포인트 지도 + 어종도감 |
| 출조 준비 | 전화 예약 + 마트 방문 | 앱 내 예약 + 채비·미끼 주문 |
| 출조 당일 | 물때 앱 따로 확인 | 물때 탭 + 출조 플래너 통합 |
| 기록·공유 | 사진 카톡 공유 | 낚시일지 + 조황 피드 업로드 |
| 커뮤니티 | 카페·단톡 분산 | 동아리·챌린지·랭킹 내재화 |
| 업체 운영 | 수동 관리 | BIZ 대시보드 + 예약 자동화 |

## 4. 타깃 고객층

### 유저 (낚시꾼)
- **연령대:** 30–50대 남성 중심, 20대·여성 낚시 인구 빠르게 증가
- **낚시 빈도:** 월 1–4회 방문하는 세미-열혈 낚시꾼
- **제주 방문 패턴:** 제주 거주자 + 내륙에서 제주 원정 낚시 오는 낚시 여행객
- **페르소나 A — 제주 거주 40대 직장인:** 주말마다 갯바위 출조, 포인트 정보·조황 피드 헤비유저
- **페르소나 B — 내륙 낚시 여행객:** 연 2–3회 제주 방문, 배편·숙소·채비를 한 번에 해결하고 싶음

### 업체 (사장님)
- **업종:** 낚시 배편 운영사, 좌대 운영업체, 낚시 전용 펜션·게스트하우스, 낚시용품점
- **규모:** 1인 또는 소규모 가족 운영 (10인 미만) 비중 약 80%
- **디지털 전환 니즈:** 예약 자동화·리뷰 관리·신규 고객 유입에 대한 필요는 있으나 직접 구축 역량 없음

## 5. 핵심 기능

| 기능 | 핵심 가치 |
|------|-----------|
| **조황 피드 + 낚시일지** | 사용자 생성 콘텐츠(UGC)가 쌓일수록 신뢰도와 SEO가 동시에 강화됨 |
| **합동출조 매칭** | 혼자 출조 비용 부담을 줄여주는 소셜 기능 → 앱 재방문율 직결 |
| **통합 예약 (좌대·배편·숙소)** | 낚시 목적에 특화된 예약 UI — 조석 시간·어종 정보 포함 |
| **채비·미끼 주문** | 출조 전 앱에서 미리 주문 → 업체 현장 픽업 또는 배송 |
| **물때 + 포인트 지도** | 제주 지역 특화 포인트 데이터 → 타 앱이 대체 불가한 핵심 락인 요소 |
| **BIZ 대시보드** | 업체의 예약 현황·매출·리뷰를 한눈에 관리 → 업체 이탈 방지 |
| **랭킹 + 챌린지** | 경쟁 심리를 자극해 조황 기록 업로드 → 트래픽 지속 생성 |

## 6. 수익 창출 모델

### 예약 수수료
- **배편 예약:** 결제액의 **7–10%**
- **좌대 예약:** 결제액의 **5–8%**
- **낚시 숙소:** 결제액의 **8–12%** (OTA 대비 낮은 수수료로 업체 유인)

### 채비·미끼 주문
- 업체 납품가 기준 **15–20% 마진** 확보
- 출조 전 패키지 구성 판매로 평균 객단가 상승 유도

### 업체 구독·광고
- **베이직 (무료):** 업체 등록 + 기본 노출
- **프리미엄 (월 39,000원):** 검색 상단 노출 + 예약 통계 + 리뷰 관리 도구
- **피처드 광고 (건당 과금):** 홈·조황 피드 내 배너 노출

### 쿠폰 프로모션
- 업체가 쿠폰 발행 시 **발행 금액의 3%** 플랫폼 수수료
- 유저 첫 예약 쿠폰 → 업체 비용 부담 없이 신규 유입 유도

### 데이터 수익화 (중장기)
- 시즌별 조황·예약 트렌드 리포트를 관광청·낚시 브랜드에 B2B 판매
- 어종 출몰 패턴 데이터 → 미끼·채비 제조사 마케팅 인사이트 제공

## 7. 영업 전략

### 초기 무료 입점 전략
- **3개월 수수료 0% 프로모션** 으로 제주 주요 낚시 업체 50곳 선입점 유치
- 업체 사장님 대상 현장 방문 온보딩 → 등록부터 첫 예약까지 대행 지원

### 업체 영업 시나리오
1. 사장님이 직접 전화·카톡으로 예약을 받는 현장 방문
2. "지금 전화 한 통에 놓치는 예약이 월 얼마인지 계산해 드릴게요" 어프로치
3. 무료 입점 + BIZ 대시보드 1주일 무료 체험 제안
4. 첫 달 예약 성사 확인 후 유료 전환 자연스럽게 연결

### KPI 지표
- **Month 3:** 입점 업체 50곳, MAU 2,000명, 예약 건수 월 300건
- **Month 6:** 입점 업체 120곳, MAU 8,000명, 월 거래액 3,000만 원
- **Month 12:** 입점 업체 200곳, MAU 20,000명, 월 거래액 1억 원

## 8. 운영 전략

### 콘텐츠 운영 (조황 피드·커뮤니티 활성화)
- 초기 시딩: 운영팀이 직접 제주 낚시 포인트 별 조황 리포트 주 3회 업로드
- **파워 유저 제도** — 월간 상위 조황 기록자에게 채비·쿠폰 리워드 지급
- 어종도감·물때 콘텐츠로 낚시 입문자 유입 → 커뮤니티 저변 확대

### 시즌별 마케팅
- **봄 (3–5월):** 갯바위·방파제 시즌 — 포인트 지도 강조 캠페인
- **여름 (6–8월):** 배낚시·가족 낚시 시즌 — 배편 예약 + 숙소 패키지 프로모션
- **가을 (9–11월):** 피크 시즌 — 랭킹·챌린지 이벤트 + 채비 할인 쿠폰
- **겨울 (12–2월):** 갈치·방어 시즌 — 합동출조 매칭 특가 캠페인

### 유저 리텐션 전략
- **출조 플래너 알림** — 물때·날씨 기반 맞춤 출조 추천 푸시 발송
- **낚시일지 스트릭** — 연속 기록 일수 뱃지·포인트 적립으로 습관 형성
- **동아리 기능** — 그룹 단위 활동으로 탈퇴 장벽 높임

## 10. 성장 로드맵

### 단기 (0–6개월) — 제주 밀착 검증
- 제주 핵심 낚시 업체 50–100곳 입점 완료
- MAU 5,000명 달성 및 조황 피드 일일 업로드 50건 유지
- 예약·주문 플로우 안정화, 결제 오류율 0.5% 미만

### 중기 (6–18개월) — 수익 구조 확립
- 유료 구독 업체 100곳 → 월 구독 매출 400만 원 확보
- 예약 GMV 월 5,000만 원 돌파 → 수수료 수익 350만 원+
- 채비·미끼 주문 월 1,000건 → 마진 매출 200만 원+
- 제주 외 도서 지역(통영·여수·완도) 확장 준비

### 장기 (18개월+) — 전국 낚시 플랫폼
- 전국 낚시 업체 1,000곳 입점 목표
- 낚시 브랜드 공식 파트너십 (채비·낚싯대 제조사) 채결
- 조황 데이터 B2B 판매 개시
- 해외 낚시 여행객(일본·중국) 대상 인바운드 관광 연계 서비스 출시

## 11. 현재 개발 현황 (MVP)

> 피치 시점 기준 — 프론트엔드 프로토타입 배포 완료

### 구현 완료 (프로토타입)

| 카테고리 | 구현된 화면 |
|----------|------------|
| **조황·정보** | 조황 피드, 조황 등록, 물때, 포인트 지도, 어종 도감, 낚시 뉴스 |
| **예약·쇼핑** | 간편 예약, 좌대, 낚시 숙소, 땡처리 렌트카, 채비·미끼 주문, 장바구니, 결제 |
| **커뮤니티** | 게시판, 글쓰기, 합동출조 매칭, 동아리, 챌린지, 랭킹 |
| **내 활동** | 마이페이지, 낚시 일지, 출조 플래너, 내 어종 분석 |
| **특별 서비스** | 훅업(매칭), 어신 AI, 선상 파티, 물반고기반 가이드 |
| **업체** | BIZ 대시보드, 업체 등록, 쿠폰 허브 |

### 기술 스택

- **프레임워크**: Next.js 15 (App Router) · TypeScript
- **배포**: AWS Lightsail · PM2 · 무중단 운영
- **디자인 시스템**: 자체 CSS 변수 기반, 라이트(Coastal) / 다크(Ocean) 듀얼 테마
- **반응형**: 모바일 우선, PC 2~3컬럼 완전 대응
- **UX**: 바텀 내비 + 전체 메뉴 드로어, 검색/알림 실시간 UI

## 13. 투자·파트너십 요청

### 시드 라운드 목표

| 항목 | 금액 |
|------|------|
| **총 모집액** | 3억 원 |
| **지분 제공** | 15~20% |
| **사용 계획** | 백엔드 개발 1억 · 영업/온보딩 1억 · 마케팅 0.5억 · 운영 0.5억 |

### 파트너십 우선 협상 대상

- **제주관광청**: 제주 낚시 관광 데이터 제공 + 공동 홍보
- **제주 낚시 업체 협회**: 초기 50개 업체 입점 패스트트랙
- **낚시 브랜드 (시마노·다이와·아부가르시아)**: 채비 공식 유통 채널
- **기상청·해양수산부**: 조류·수온 실시간 데이터 API 협약

1. **낚시 인구 급증**: 코로나 이후 국내 낚시 인구 900만 돌파, 연 10%+ 성장

## 14. QA 진행 방향

### 유저 플로우 테스트 시나리오
- [ ] 회원가입 → 조황 피드 조회 → 조황 등록 (사진 첨부 포함)
- [ ] 배편 검색 → 날짜·인원 선택 → 결제 → 예약 확인 알림 수신
- [ ] 합동출조 글 등록 → 다른 계정으로 신청 → 매칭 확정 알림
- [ ] 채비·미끼 주문 → 장바구니 → 결제 → 주문 내역 확인
- [ ] 물때 탭에서 날짜 변경 → 포인트 지도 특정 위치 클릭 → 조황 정보 확인
- [ ] 낚시일지 작성 → 공개/비공개 설정 → 피드 노출 여부 확인

### 업체 플로우 테스트 시나리오
- [ ] 업체 등록 → 업종 선택 → 메뉴/프로그램 등록 → 노출 확인
- [ ] 업체 대시보드 로그인 → 예약 요청 확인 → 승인·거절 처리
- [ ] 쿠폰 생성 → 유저 쿠폰 적용 결제 → 정산 내역 반영 확인
- [ ] 채비 상품 등록 → 유저 주문 → 업체 주문 알림 수신
- [ ] 리뷰 등록 (유저) → 업체 답변 등록 → 노출 순서 확인
