# 렌터카 x 파티 통합 전략

## 핵심 아이디어

> 파티가 렌터카를 팔고, 렌터카가 파티를 만든다

```
[순방향] 파티 생성 → "렌터카 같이 빌려요" → 엔빵 예약
[역방향] 렌터카 예약 → "이 차로 참여 가능한 파티" 추천
[패키지] 렌터카 + 카페패스 + 파티 = 올인원 번들
```

---

## 1. 파티 내 렌터카 연계

### "렌터카 같이 빌려요" 핵심 UX

파티 만들기에서 교통수단 3단 선택:

| 선택지 | 설명 | 추가 UI |
|--------|------|---------|
| 🚶 각자 이동 | 렌터카 불필요 | 없음 |
| 🚗 내 차로 | 본인 차 동승 제공 | 차종 텍스트 입력 |
| 🚗+ 같이 빌려요 | **플랫폼 렌터카 엔빵** | 차종 선택 → 자동 엔빵 계산 |

### 엔빵 효과 (가격 장벽 파괴)

| 차종 | 일일가 | 1인 | 2인 엔빵 | 4인 엔빵 | 7인 엔빵 |
|------|--------|-----|---------|---------|---------|
| 레이 | 33,000원 | 33,000원 | 16,500원 | **8,250원** | - |
| 투싼 | 59,000원 | 59,000원 | 29,500원 | **14,750원** | - |
| 카니발 | 89,000원 | 89,000원 | 44,500원 | 22,250원 | **12,714원** |

> 카니발 7인 엔빵 = **커피 3잔 가격**으로 하루 차량 이동

### 카테고리별 추천 차종

| 카테고리 | 렌터카 필요도 | 이유 | 추천 차종 |
|---------|------------|------|---------|
| cycling | 높음 | 자전거 운반 | 카니발, 팰리세이드 |
| hiking | 매우 높음 | 새벽출발, 산간 | 투싼, 팰리세이드 |
| surfing | 높음 | 서핑보드 적재 | 카니발 |
| cafe | 중간 | 카페 간 이동 | 레이, 아반떼 |
| drive | 필수 | 드라이브 자체가 목적 | 쏘나타, EV6 |
| fishing | 높음 | 장비 운반 | 카니발 |

### 참여 모달 비용 표시

```
파티비                    15,000원
렌터카 엔빵 (투싼 ÷ 4명)   14,750원
─────────────────────────
합계                     29,750원
+ 카페패스 14,900원~ (선택)
```

참여자가 "렌터카 동승" vs "각자 이동" 선택 가능

### 피드 카드 뱃지 변경

```
기존: [🚗 렌터카]
변경: [🚗 엔빵 14,750원]  ← 가격이 바로 보임
```

## 2. 역방향 — 렌터카가 파티를 만드는 모델

### 렌터카 예약 페이지에서 파티 추천

```
[AI 추천 결과: 투싼 SUV]

🎉 이 차로 참여 가능한 파티
━━━━━━━━━━━━━━━━━━━━━
⛰️ 한라산 영실코스 (4/13) — 동승 2자리
🚴 애월 자전거 라이딩 (4/12) — 자전거 적재 가능
```

### 렌터카 예약자에게 알림톡

```
[카카오 알림톡]
{이름}님, 4/12 투싼 예약 확정!

🎉 이 기간 파티가 있어요:
- 애월 자전거 라이딩 (4/12) — 차에 자전거 싣고!
- 한라산 등산 (4/13) — 새벽 출발 차 필수!

참여하면 렌터카비 엔빵 가능 💰
```

### 드라이브 파티 코스 (카페투어 코스와 동일 구조)

- 동쪽 해안: 함덕 → 김녕 → 월정리 → 세화 → 성산 (45km, 4시간)
- 서쪽 해안: 애월 → 한림 → 협재 → 중문 (50km, 4시간)
- 중산간: 1100도로 → 비자림 → 사려니숲 (35km, 3시간)

## 5. 구현 우선순위

### Week 1~2
1. Party 타입에 렌터카 필드 추가 (rentalCarMode, rentalCarId, etc.)
2. car-data.ts (차종 데이터) 공유 파일 생성
3. 파티 생성 폼: 3단 교통수단 선택 + 차종 그리드 + 엔빵 계산기
4. 더미 데이터 업데이트

### Week 3~4
5. 파티 상세: 렌터카 정보 카드 + 좌석 시각화 (●●○○)
6. 참여 모달: 동승/각자이동 선택 + 비용 분리 표시
7. 피드 뱃지: "🚗 엔빵 14,750원"
8. car-pick에서 관련 파티 추천

### Week 5~6
9. 결제 통합 (파티비 + 렌터카비 합산)
10. 드라이브 코스 데이터 생성
11. 패키지 랜딩 페이지
12. 렌터카 예약자 → 파티 추천 알림
