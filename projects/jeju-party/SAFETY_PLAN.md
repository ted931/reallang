# 제주 취미 파티 — 신뢰 & 안전 기획서

## 핵심 문제
낯선 사람끼리 만나는 번개 모임 → 인증 없으면 위험, 인증 너무 무거우면 이탈

## 비용 문제 (엔빵)
| 시나리오 | 피해자 | 해결책 |
|---------|--------|--------|
| 파티장이 예약 후 참여자 노쇼 | 파티장 | 에스크로 선결제 + 노쇼 패널티 |
| 파티장이 돈 걷고 잠수 | 참여자 | 에스크로 (플랫폼 예치) + 자동 환불 |
| "나중에 걷을게" 했는데 안 냄 | 파티장 | 사전 결제 필수화 |

### 에스크로 모델
```
참여 신청 → 플랫폼에 결제 (파티장에게 안 감)
→ 파티 완료 → 24시간 후 자동 정산
→ 노쇼/취소 → 환불 규정 자동 적용
→ 파티장 잠수 → 전액 자동 환불
```

### 환불 규정
- 파티 48시간 전: 전액 환불
- 파티 24시간 전: 50% 환불
- 당일: 환불 불가 (파티장 취소 시에는 전액 환불)

## Phase 1 (MVP) — 즉시 구현

### P0: 법적 필수
- [ ] 이용약관 페이지 (`/terms`)
- [ ] 개인정보처리방침 페이지 (`/privacy`)
- [ ] Footer에 사업자정보 + 약관 링크
- [ ] 참여/생성 시 만14세 이상 + 개인정보 동의 체크박스
- [ ] 결제 시 환불 규정 고지

### P0: 본인인증
- [ ] 휴대폰 SMS 인증 (첫 참여/생성 시 1회)
- [ ] 인증 완료 뱃지 (호스트 프로필에 "📱 인증됨")
- [ ] 미인증 시 참여/생성 버튼 비활성화

### P0: 에스크로 결제
- [ ] 참여 시 플랫폼에 결제 (파티장 직접 수령 X)
- [ ] 파티 완료 후 자동 정산
- [ ] 취소/노쇼 시 환불 규정 자동 적용
- [ ] 파티장 취소 시 전액 환불

### P1: 신고/차단
- [ ] 파티 상세에서 신고 버튼 (노쇼/사기/불쾌/안전우려)
- [ ] 사용자 차단 (차단한 사람의 파티 피드에서 숨김)
- [ ] 신고 3건 누적 시 자동 비노출

### P1: 안전 장치
- [ ] 안전 공유 링크 (참여 후 카카오톡으로 파티 정보 공유)
- [ ] 긴급연락처 등록 (선택, 강력 권장)
- [ ] 장소 경고 ("집/숙소" 키워드 감지 시 공공장소 권장)
- [ ] 파티 생성 시 안전 가이드라인 표시

## Phase 2 — 신뢰 기반

### 인증 고도화
- [ ] 카카오 로그인 통합
- [ ] 호스트 승인제 (참여 신청 → 호스트가 프로필 보고 승인/거절)

### 후기/평가
- [ ] 파티 후 양방향 리뷰 (호스트↔참여자)
- [ ] 별점 + 태그 (#친절 #시간엄수 #재밌음 #또참여하고싶어요)
- [ ] 동시 공개 (양쪽 다 작성해야 공개 → 보복 리뷰 방지)

### 신뢰 뱃지
- [ ] 📱 인증됨 (SMS 인증)
- [ ] ⭐ 후기 10+ (리뷰 10개 이상)
- [ ] 🏅 슈퍼호스트 (개최 10회+ & 평점 4.5+)
- [ ] ✅ 무노쇼 (참여 5회+ & 노쇼 0회)

### 안전 귀가
- [ ] 파티 종료 후 "무사히 귀가했나요?" 체크인
- [ ] 2시간 미응답 시 긴급연락처 알림

## 인증 단계별 전략

| 행동 | 요구 수준 | 근거 |
|------|----------|------|
| 피드 구경 | 없음 | 유입 유지 |
| 무료 파티 참여 | SMS 인증 | 최소 신원 확인 |
| 유료 파티 참여 | SMS 인증 | 결제 시 자연스럽게 |
| 파티 생성 | SMS + 프로필 필수 | 호스트는 더 높은 책임 |
| 유료 파티 생성 | SMS + 첫 파티 이력 | 사기 방지 |

## 디자인 원칙
- 안전 컬러: emerald(신뢰), red(긴급), amber(주의)
- 인증은 한 번만, 이후 세션 유지
- 참여 버튼 → 인증 → 참여 완료까지 최대 3탭
- 안전 기능은 무대 뒤에서 작동, 필요할 때만 앞으로
