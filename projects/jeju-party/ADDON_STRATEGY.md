# 렌터카 사이트 Add-on 연동 전략 + 알림 여정 설계

## 핵심 전략

> 렌터카 예약 플로우의 **모든 접점**에서 카페패스/파티/코스 가이드를 노출
> 카페패스 번들은 **미끼 상품** (적자 감수) → 렌터카 전환율 + 재방문으로 회수

---

## 1. 예약 플로우 내 Add-on 삽입 포인트

### Step 1: 차량 옵션 선택 (보험/네비 옆)

```
━━━ 차량 옵션 ━━━━━━━━━━━━━━━━━━━━━
☑ 완전자차보험          +15,000원/일
☐ 네비게이션                   무료
☐ 카시트                +5,000원/일

━━━ 🏝️ 제주 여행 Add-on (NEW) ━━━━━
☐ 카페패스 3일 프리미엄   10,430원   [30% OFF] [인기]
  └ 50+ 제휴 카페에서 3시간마다 음료 무제한
☐ 카페패스 5잔권          12,400원   [20% OFF]

📍 날씨 맞춤 드라이브 코스 가이드 [무료 받기→]
```

- 전환율: 카페패스 **8~12%**, 코스 가이드 **35~45% 클릭**
- 여행 기간에 맞는 패스 자동 추천 (3일 이하→3잔권, 3일→3일권, 4일+→5일권)

### Step 2: 결제 직전 (파티 추천 + 카페패스 리마인더)

```
━━━━━━━━━━ 결제 요약 ━━━━━━━━━━

렌터카: 투싼 3일              177,000원
보험: 완전자차                 45,000원
카페패스 3일 프리미엄           10,430원  [✕ 제거]
───────────────────────────────
합계                         232,430원
✅ 카페패스 포함으로 약 4,500원 절약!

🎉 4/25~27 참여 가능한 파티
[⛰️ 한라산 영실코스]  [☕ 애월 카페 투어]
 4/25 07:00           4/26 10:00
 엔빵 ~25,000원       엔빵 ~15,000원
```

- 파티: 예약 날짜+인원+차종으로 자동 매칭, 전환율 **3~5%**
- 카페패스 미선택자: "지금 추가하면 30% 할인" 배너, 추가 전환 **2~3%**

### Step 3: 결제 완료 (확인 페이지)

- 카페패스 미구매자: 풀폭 프로모 배너, 전환 **5~8%**
- 드라이브 코스 가이드: [카카오톡으로 받기], 전환 **40~55%**
- 파티 추천 카드, 전환 **4~6%**
- 체험/액티비티 쿠폰 (서핑/스노클링/공방), 전환 **3~5%**

### Step 4: 차량 인수 (현장 QR)

차 키 봉투에 QR 스티커 → 모바일 랜딩:

```
🏝️ {이름}님의 제주 여행 가이드

📍 오늘의 추천 코스 (실시간 날씨 기반)
☕ 카페패스 — 현장 전용 40% 할인 (미구매자)
🎉 오늘 참여 가능한 파티
🔧 스마트 주유 / 긴급 연락처
```

- QR 스캔율 **60~70%**, 카페패스 전환 **10~15%**
- 현장 40% 할인 = 미끼 (건당 -3,560원 적자, 투자비)

### Step 5~6: 여행 중 + 반납 후 → 알림 여정 (아래 참조)

## 5. 기술 연동

### 데이터 공유

```
rentcar.jejupass.com ←→ Supabase (공유 DB) ←→ realang.store/party
```

- 예약 정보(날짜, 인원, 차종) → 파티 매칭 API
- 카페패스 구매 상태 → 알림톡 분기 (구매자/미구매자)
- QR 랜딩 → 실시간 날씨 + 카페패스 상태

### 결제 방식

| 상품 | 결제 | 이유 |
|------|------|------|
| 렌터카 + 카페패스 | **통합** | 번들 할인, 원클릭 전환 |
| 파티 참여비 | **분리** (에스크로) | 취소/노쇼 대응 |
| 체험/액티비티 | **분리** (외부 링크) | CPA 제휴사 결제 |

### 계정 통합 로드맵

| Phase | 방식 |
|-------|------|
| 1 | URL 파라미터 + JWT 토큰 전달 |
| 2 | Supabase Auth (카카오 로그인 공유) |
| 3 | 서브도메인 통합 (car/party/cafe.jejupass.com) |

## 6. 신규 vs 재방문 고객 차이

| | 신규 | 재방문 |
|--|------|--------|
| 톤 | 친절 안내형, 제주 팁 | "다시 만나서 반가워요!" |
| 크로스셀 | 카페패스 위주 (저가) | 올인원 패키지 (고가) |
| 할인 | 카페패스 30% | 렌터카 VIP 10% + 패키지 25% |
| 파티 | "이런 것도 있어요" 소개 | 이전 참여 기반 맞춤 추천 |
| 리뷰 | 간단 별점 | 상세 리뷰 + 사진 (인센티브↑) |
