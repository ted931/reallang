## 1. 한 줄 요약

> **제주도 여행자가 '검색 → 동행 → 이동 → 결제'까지 이탈 없이 경험하도록 설계된 디지털 에코시스템으로, 연간 1,500만 여행객을 렌터카·카페패스·프리미엄 가게 등록 매출 퍼널로 전환한다.**

## 2. 문제 정의

### 여행자 관점

제주도 여행을 계획하는 사람은 평균적으로 5~7개의 앱과 웹사이트를 동시에 열고 정보를 조합한다. 이 과정에서 겪는 핵심 불편은 다음과 같다.

| 단계 | 불편 |
|------|------|
| 계획 | 맛집·카페 정보가 네이버 블로그, 인스타그램, 카카오맵에 분산되어 있어 신뢰도가 낮고 최신성이 불분명함 |
| 동행 | 혼자 여행하거나 소규모 그룹일 경우 같은 취향의 동행자를 찾을 플랫폼이 없음 |
| 이동 | 렌터카 비교·예약이 복잡하고, 날씨·코스에 맞는 차종 추천이 없음 |
| 발견 | 여행 중 방문한 가게가 제주패스 카페패스 가맹점인지 알 방법이 없음 |
| 리뷰 | 방문 후 경험 공유 채널이 분산되어 있어 재방문·추천 루프가 형성되지 않음 |

**핵심 문제**: 정보·예약·결제가 분산된 탓에 여행자는 제주패스의 핵심 상품(렌터카, 카페패스)을 인지하거나 구매할 기회를 놓친다.

### 사업자(가게) 관점

제주 등록 음식점·카페 약 2만 개 중 디지털 채널을 주체적으로 운영하는 곳은 20% 미만으로 추산된다. 나머지는 플랫폼 의존적이거나 아무런 디지털 존재감이 없다.

| 단계 | 불편 |
|------|------|
| 홍보 | 네이버 플레이스, 인스타그램, 카카오 등 각 채널에 별도로 정보를 입력·관리해야 함 |
| 예약 | 전화 예약 의존도가 높고, 노쇼 문제 해결 수단이 없음 |
| 분석 | 방문 고객 데이터가 없어 마케팅 의사결정이 감에 의존함 |
| 업셀 | 관광객 대상 패키지(카페패스 등) 연동 방법을 모름 |
| 비용 | 쿠팡이츠·배달의민족 등 기존 플랫폼의 수수료(15~35%)가 과도함 |

**핵심 문제**: 사업자가 디지털 채널에 진입하는 비용과 복잡성이 높아, 제주패스와의 협력 기회를 인지·활용하지 못한다.


### 4-1. jejupass-promo (포트 3001)

#### 핵심 가치

제주 여행 전 "어떤 카페, 어떤 맛집이 진짜 좋은 곳인지" 신뢰할 수 있는 정보를 원하는 여행자에게 검증된 가게 정보를 제공한다. 동시에 사업자에게는 무료로 디지털 존재감을 확보할 수 있는 진입 채널을 열어준다. 이 앱의 모든 SEO 트래픽은 최종적으로 렌터카 예약과 카페패스 구매로 연결되도록 설계되었다.

#### 사용 시나리오

**여행자 시나리오**
- 서울 거주 직장인 A씨(30대)가 제주 여행 2주 전 "제주 카페 추천"을 검색한다.
- `/explore/cafe` 페이지가 검색 결과 상위에 노출되고, A씨는 카테고리별 필터로 분위기·지역을 좁혀간다.
- 마음에 드는 카페 상세 페이지에서 메뉴, 사진, 리뷰, 영업시간, 지도를 확인한다.
- 이동 수단이 필요하다는 것을 인지하고 `/rentcar`로 이동, AI가 3박 4일 여행에 최적인 차종을 추천해준다.
- 렌터카 예약 완료 → 카페패스 업셀 CTA 노출.

**사업자 시나리오**
- 제주 서귀포의 베이커리 사장 B씨가 "카카오맵에만 있는 내 가게를 더 홍보하고 싶다"고 생각한다.
- 지인으로부터 jejupass-promo 무료 등록 링크를 받아 5분 만에 가게 정보를 입력한다.
- 방문자 통계가 쌓이기 시작하고, 한 달 후 프리미엄 업그레이드 안내를 받는다.
- 프리미엄으로 전환하면 카페패스 가맹 가입 안내가 제공된다.

#### 주요 기능

| 기능 | 설명 |
|------|------|
| 카테고리 탐색 | 카페, 맛집, 베이커리, 브런치, 바 등 카테고리별 필터링 |
| 가게 상세 | 메뉴, 사진, 리뷰, 영업시간, 위치 지도, SNS 연동 |
| SEO 최적화 페이지 | `/explore/{category}` 형태의 정적 카테고리 페이지 |
| 렌터카 페이지 | AI 차종 추천, 8종 비교표, 예상 비용 계산기 |
| 사업자 등록 | 무료 가게 등록 → 인증 → 관리 대시보드 (jeju-biz 연동) |
| 해시태그 자동생성 | 가게 정보 기반 SNS 해시태그 자동 생성 |

#### 이 앱이 매출에 기여하는 방식

1. **SEO 트래픽 → 렌터카 전환**: "제주 카페" 등 여행 전 검색 유입 → 렌터카 페이지 이동 → jejupass.com 예약 커미션
2. **SEO 트래픽 → 카페패스 전환**: 여러 카페 탐색 중 카페패스 CTA 노출 → 구매 전환
3. **사업자 등록 → 프리미엄 수익**: 무료 등록 후 프리미엄 업그레이드 월정액 수익

### 4-2. jeju-party (포트 3010)

#### 핵심 가치

혼자 또는 소규모로 제주에 오는 여행자의 "취향 맞는 동행을 찾고 싶다"는 니즈를 충족한다. 파티(동행 그룹) 형성 과정에서 jeju-pass 등록 가게를 코스에 자연스럽게 포함시키고, 렌터카·카페패스 구매를 이동과 소비의 흐름에서 유도한다. 소셜 경험 자체가 제주패스 브랜드와 결합되는 구조다.

#### 사용 시나리오

**개인 여행자 시나리오**
- 제주 혼자 여행 예정인 C씨(20대)가 서핑 동행을 찾고 있다.
- jeju-party에서 "서핑 파티"를 검색, 비슷한 날짜·연령대의 파티를 발견하고 참가 신청을 한다.
- 파티 코스에 제주패스 등록 카페가 경유지로 포함되어 있다.
- 파티 리더가 렌터카 공유를 제안, `/rentcar`로 이동하여 7인승 차량을 예약한다.
- 파티 종료 후 카페패스 업셀 화면이 노출된다.

**AI 일정 생성 시나리오**
- D씨는 3박 4일 제주 여행 계획을 AI에게 요청한다.
- AI가 하이킹 코스 3가지를 제안하고, 각 코스에 맛집·카페 경유지가 포함된다.
- D씨가 마음에 드는 코스를 선택하면 일자별 타임라인이 완성된다.
- "이 일정으로 파티 만들기" 버튼으로 동행 모집 파티가 즉시 생성된다.
- 코스에 포함된 가게들은 모두 jejupass-promo 등록 가게로 노출 효과를 얻는다.

#### 주요 기능

| 기능 | 설명 |
|------|------|
| 파티 생성·탐색 | 활동 유형, 날짜, 인원, 지역별 파티 생성 및 참가 신청 |
| 활동 카테고리 | 하이킹, 서핑, 카페투어, 요리 클래스, 자전거, 스노클링 등 |
| 코스 연동 | 파티 코스에 jejupass 등록 가게 경유지 포함 |
| AI 일정 생성 | 여행 스타일·기간 입력 → 3가지 코스 추천 → 일자별 타임라인 |
| 파트너 배너 | 렌터카·카페패스 파트너 배너 노출 |
| 업셀 CTA | 파티 참가 완료 후 렌터카·카페패스 구매 유도 화면 |

#### 파티 유형 — 두 가지 트랙

jeju-party는 성격이 다른 두 가지 파티 트랙을 동시에 운영한다.

| 구분 | 개인 파티 (비용분담형) | 사업자 파티 (유료 판매형) |
|------|---------------------|----------------------|
| 주최자 | 일반 여행자 | 인증 사업자 (서핑 스쿨, 가이드 등) |
| 비용 | 실비 n/1 분담 | 좌석 유료 판매 (선입금) |
| 플랫폼 수수료 | 없음 | 결제액의 12% |
| 목적 | 동행·경험 공유 | 체험 상품 판매 |
| 예시 | "같이 한라산 갈 사람" | "서핑 입문 클래스 6명 모집" |

**개인 파티**는 수수료가 없는 대신 렌터카·카페패스 업셀의 트래픽을 만든다. **사업자 파티**는 결제를 플랫폼이 경유하여 12% 커미션을 수취하고, 행사 완료 후 업체에 정산한다.

#### 사업자 파티 사용 시나리오

> **김서퍼 씨 (서핑 스쿨 운영)**: 매주 토요일 협재 해변에서 서핑 입문 클래스 운영. 기존엔 인스타그램 DM으로 예약받고 노쇼 시 환불 분쟁 발생. jeju-party 사업자 파티 등록 후 선입금 자동화 → 노쇼 0건. 월 20개 클래스 × 6명 × 65,000원 = 780만원 매출, 플랫폼 수수료 12% = 93,600원.

#### 이 앱이 매출에 기여하는 방식

1. **파티 참가 후 렌터카 업셀**: 여러 명이 함께 이동하는 파티 특성상 렌터카 전환율이 일반 대비 높음 (그룹 이동 수요)
2. **코스 내 카페 경유 → 카페패스 구매**: 카페패스 가맹 카페 방문 경험 후 구매 전환
3. **사업자 파티 커미션 (12%)**: 서핑·가이드·체험 업체의 유료 좌석 판매 결제 중개 수수료
4. **개인 파티 참가비 커미션 (10%)**: 소규모 유료 개인 파티 참가비 수수료
5. **AI 코스 공유 바이럴**: 코스 공유 시 jejupass-promo 가게들이 자동 노출 → SEO·브랜드 효과

### 4-3. jeju-map (포트 3005)

#### 핵심 가치

제주 현지에 있는 여행자가 "지금 근처에 뭐가 있지?"를 가장 빠르게 확인할 수 있는 지도 기반 탐색 도구다. 실시간 날씨 정보와 결합하여 "오늘 날씨에 맞는 곳"을 추천하고, 미등록 가게를 발견했을 때 제주패스 등록 CTA를 즉시 노출한다. 현지 체류 중 소비 결정에 직접 개입하는 앱이다.

#### 사용 시나리오

**현지 탐색 시나리오**
- 현지에 도착한 E씨가 jeju-map을 열고 현재 위치 기반으로 근처 카페를 찾는다.

**날씨 기반 코스 시나리오**

### 4-4. jeju-biz (포트 3022)

#### 핵심 가치

가게 사업자가 디지털 홍보·예약·분석을 한 곳에서 처리할 수 있도록 한다. 진입 장벽을 낮춰 무료로 시작하게 하고, 데이터가 쌓이면서 유료 전환의 가치를 스스로 느끼도록 설계한다. 카페패스 가맹과 파티 경유지 등록은 이 앱에서 신청한다.

#### 사용 시나리오

**신규 사업자 시나리오**
- jejupass-promo에서 가게를 등록한 F 사장이 인증 완료 후 jeju-biz 대시보드로 이동한다.
- 오늘 방문자 수, 페이지뷰, 문의 수가 실시간으로 집계된다.
- `/promo` 페이지에서 무료 홍보 페이지(카드형 랜딩 페이지)를 5분 만에 생성한다.
- SNS 공유용 링크와 QR코드가 자동 생성된다.

**파트너 오퍼 시나리오**
- `/partner` 페이지에서 카페패스 가맹 신청 버튼을 클릭한다.
- 가맹 조건(최소 판매 수량, 단가 설정 등)을 확인하고 신청서를 제출한다.
- 가맹 승인 후 카페패스 판매 현황이 대시보드에 통합 표시된다.
- 월 수익 정산 내역이 `파트너 오퍼 관리` 탭에서 확인된다.

#### 주요 기능

| 기능 | 설명 |
|------|------|
| 예약 현황 | 실시간 예약 접수·확인·취소 관리 |
| CS 티켓 | 고객 문의 인입·처리·답변 관리 |
| 매출 통계 | 방문자, 전환, 매출 일별/월별 분석 |
| 홍보 페이지 | 무료 카드형 랜딩 페이지 생성 및 QR코드 발행 |
| 파트너 오퍼 | 카페패스 가맹, 파티 경유지 등록, 렌터카 제휴 신청 |
| 업셀 CTA | 홍보 완료 후 카페패스·렌터카·파티 연동 제안 |

#### 이 앱이 매출에 기여하는 방식

1. **프리미엄 가게 등록 전환**: 무료 대시보드 사용 후 유료 기능(상단 노출, 예약 연동) 전환 유도
2. **카페패스 가맹 확대**: 파트너 오퍼를 통해 카페패스 가맹 가게 수 증가 → 카페패스 상품 가치 상승
3. **사업자 LTV 확대**: 하나의 사업자가 홍보 페이지 → 프리미엄 → 카페패스 → 광고 순서로 ARPU 상승

- 목표: 6개월 내 월 유기 검색 유입 3만 명 확보

## 7. 경쟁 우위

### 현재 경쟁 환경

| 경쟁사 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|------------|
| 네이버 플레이스 | 압도적 트래픽, 지도 연동 | 제주 특화 없음, 여행 의도 없음 | 제주 여행 여정 전체 커버 |
| 카카오맵 | 국민 앱 수준 인지도 | 예약·결제·동행 기능 없음 | 렌터카·카페패스 통합 결제 |
| 야놀자 / 여기어때 | 숙박 강자, 대규모 트래픽 | 제주 현지 가게 탐색 약함 | 현지 가게 특화, 사업자 직접 채널 |
| 에어비앤비 체험 | 글로벌 브랜드, 체험 상품 | 한국 현지화 약함, 수수료 높음 | 제주 특화, 낮은 수수료, 국내 서비스 |
| 망고플레이트 / 식신 | 음식 리뷰 DB 풍부 | 예약·결제·이동 연동 없음 | 탐색부터 렌터카·패스까지 원스톱 |

### 핵심 차별점 3가지

**1. 제주 여행 여정의 수직 통합**

경쟁사들은 탐색(네이버), 숙박(야놀자), 이동(없음), 동행(없음) 각각에 특화되어 있다. 제주 디지털 에코시스템은 탐색 → 동행 → 이동 → 현지 소비의 전 여정을 단일 브랜드(제주패스) 아래 통합한다. 여행자가 이탈하지 않고 구매까지 도달하는 최단 경로를 제공한다.

**2. 사업자와 여행자의 양면 시장 동시 구축**

대부분의 경쟁사는 여행자(소비자) 중심이다. 제주 에코시스템은 사업자(jeju-biz)에게 무료 디지털 툴을 제공하여 사업자를 플랫폼 내에 묶어두는 동시에, 여행자에게 더 풍부한 가게 데이터를 제공하는 양면 플라이휠을 형성한다.

**3. AI 기반 개인화 + 제주 특화 데이터**

일반 여행 플랫폼은 전국 데이터를 다루므로 제주 특화 AI 모델을 학습시키기 어렵다. 제주패스는 제주 렌터카 데이터, 카페패스 이용 데이터, 가게 정보를 이미 보유하고 있어, 제주 날씨·계절·이벤트에 특화된 AI 추천 품질이 경쟁사 대비 높다.
